Descripcion

Como hacer solicitudes webs en python, ya sea para descargar una página o solicitar información a un endpoint.

Metodo

Para realizar solicitudes oslo tenemos que importar el modulo request y utilizarlo de la siguiente manera:

import requests

response = requests.get("http://time.jsontest.com")

print(response.text)
