Compilador del Framework nativo de Windows

Windows por defecto trae integrado .Net framework 4, el compilador se encuentra en la siguiente ruta:

C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc

Para compilar un programa simplemente usamos la ruta absoluta al csc y le pasamos el archivo a compilar:

> C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc Program.cs
Compilacion con .Net

Para compilar un proyecto con .Net podemos hacer lo siguiente:

Creamos un proyecto base:

> dotnet new console

Compilamos el proyecto:

> dotnet build

Ejecutamos el proyecto:

> dotnet run
Distintos tipos de .Net

En .Net encontramos varios frameworks:

  • .Net Framework
  • Mono
  • .Net Core
