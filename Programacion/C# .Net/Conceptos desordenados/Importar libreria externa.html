Como importar clase externa en un proyecto de .Net

Metodo

Partimos de una clase como la siguiente:

Ahora crearemos una librería externa a este proyecto que importaremos despues para ser utilizada en el proyecto Calculator.

Nos vamos a New > Project

Buscamos Class Library y seleccionamos el template de Class Library:

Damos nombre a la librería que vamos a crear y añadimos el proyecto a nuestra solucion:

Definimos el código de nuestra librería externa:

Usamos nuestra librería externa en nuestro proyecto principal:

Como podemos ver se producen unos errores en el código, esto se debe a que no tenemos la libería externa referenciada:

Nos vamos a Project > Add Project Reference y añadimos nuestra librería externa.

Ya podemos usar el código de la librería externa en nuestro proyecto

Si hacemos doble click encima de proyecto Calculator se abre el archivo del proyecto:

En dicho archivo podemos ver la importación de la libería externa.

NOTA: Si intentamos abrir el archivo del proyecto en un proyecto con .Net Framework 4.x el visual studio no nos abrirá el archivo, si quremos ver el archivo tenemos que abrirlo de manera externa con el notepad o algun otro editor.
