Simplificar Console (static imports)

Mbascoy.dev / C# .Net / Simplificar Console (static imports)
Descripción

Como utilizar las importaciones estáticas para simplificar el uso de algunas clases.

Metodo

Si tenemos un código como el siguiente:

Console.WriteLine("Hola Mundo");

Podemos prescindir del Console de la siguiente manera:

using static System.Console;

WriteLine("Hola Mundo");
