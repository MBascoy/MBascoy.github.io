Descripcion

Diferencias entre public, private, var y let.

Metodo

La diferencía entre public/private y var/let es el ambito donde se usan.

Public/private se utiliza en la declaración de clases e indican desde donde es alcanzable la variable, public desde cualquiera otra clase y private solo desde la propia clase

Var/let se utilizan en los propios bloques de ejecución de código, las variables let son solo accesibles dentro del mismo bloque de código y las variables var desde cualquiera otro bloque de código
