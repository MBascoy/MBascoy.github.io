Swagger issue NumberFormatException

Mbascoy.dev / Spring / Swagger issue NumberFormatException
Descripcion

Al ejecutar la pagina de swagger-ui.html se proudce una NumberFormatException:

El problema se debe a un fallo de swagger cuando se intenta documentar una variable que se tiene que pasar como un entero y swagger espera que se pase como un string

Solucion

Para solucionar este error lo unico que tenemos que hacer es indicar el parametro example dentro de los @ApiParam, por ejemplo:

@ApiParam(value = "[Optional] idLanguage param for language query.", example = "12") @RequestParam(required = false) Long idLanguage,

El valor que se le da al example en este caso tendria que ser un valor numerico ya que el parametro que se esta documentando es un long.
