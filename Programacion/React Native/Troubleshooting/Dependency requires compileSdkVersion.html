Dependency requires compileSdkVersion

Mbascoy.dev / React Native / Dependency requires compileSdkVersion
Descripcion

Al intentar compilar una aplicación obtenemos el siguiente error:

Dependency 'androidx.appcompat:appcompat-resources:1.5.0' requires 'compileSdkVersion' to be set to 32 or higher.
Solucion

Para solucionar este problema tenemos que cambiar la versión que le estámos indicando a gradle del appcompat.

Para ello tenemos que editar el archivo build.gradle que está dentro de la carpeta app.

NOTA: hay varios archivos build.gradle en un proyecto, el que buscamos en este caso es el que está en: android/app/build.gradle

Abrimos el archivo y editamos la siguiente linea (está casi al final del todo):

implementation 'androidx.appcompat:appcompat:1.5.0'

Y cambiamos la versión del Appcompat para poner la 1.4.1, la línea quedaría tal que así:

implementation 'androidx.appcompat:appcompat:1.4.1'

Una vez hechos los cambios guardamos el archivo y volvemos a reconstruir el proyecto, Build > Make Project.
