Declarar clase solo para constantes

Mbascoy.dev / Angular / Declarar clase solo para constantes
Descripcion

Crear una clase a la que llamaremos cuando queramos acceder a constantes de la aplicación.

Metodo

Para realizar esto solo tenemos que crear una clase normal y a continuación declararemos las constantes como estáticas.

Ejemplo:

export class AppConstants {
  public static readonly urlBack = "example.com";
  public static readonly maxNumberOfUsers = 100;
  public static readonly appName = "example";
}

Una vez tenemos esta clase podemos llamar a las constantes desde cualquier parte usando:

AppConstants.maxNumberOfUsers

(Necesitamos importar dicha clase en los sitios donde vayamos a llamar a la clase de constantes.)
