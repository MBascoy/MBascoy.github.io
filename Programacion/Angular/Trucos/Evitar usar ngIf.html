Evitar usar ngIf

Mbascoy.dev / Angular / Evitar usar ngIf
Descripcion

Si queremos prescindir de utilizar ngIf podemos utilizar la siguiente alternativa

Simplemente utilizamos la directiva [hidden] y una variable booleana para determinar si mostrar un elemeneto o no:

<p>
    <a [hidden]="hideSeeMore" (click)="toggle()" data-bs-toggle="collapse" href="#multiCollapseExample{{memberId}}" role="button" aria-expanded="false" aria-controls="multiCollapseExample1">{{see}}</a>
</p>

En este caso usamos la variable hideSeeMore para controlar si se muestra la etiqueta o no

Y en el ts controlamos la variable de la siguiente manera:

  ngOnInit(): void {
    if(this.skillList?.length < 2)
      this.hideSeeMore = true;
    else
      this.hideSeeMore =  false;
  }
