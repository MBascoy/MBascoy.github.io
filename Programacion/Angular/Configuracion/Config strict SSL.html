Como configurar npm para que no sea estricto con los certificados SSL

A veces al intentar instalar un paquete con npm podemos obtener un error como:

UNABLE_TO_GET_ISSUER_CERT_LOCALLY
Metodo

Para solucionar este problema tenemos que configurar npm para que no sea stricto con los certificados ssl:

npm config set strict-ssl false

y a mayores tenemos que usar la URL del repositorio con http en vez de https
