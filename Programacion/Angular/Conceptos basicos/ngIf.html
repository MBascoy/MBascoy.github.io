Como utilizar la sentencia de control ngIf para controlar que etiquetas mostrar

Descripcion

Para utilizar el ngIf lo podemos hacer de la siguiente manera:

Teniendo el siguiente componente:

import { Component } from '@angular/core';

@Component({
    selector: 'empleado',
    templateUrl: './empleados.component.html'
  })

export class empleadosComponent {
    nombre:string = "Pepe";
  }

Podemos usar la directiva *ngIf dentro de una etiqueta para decidir si se muestra o no:

<h1 *ngIf="nombre=='Pepe'">el nombre es Pepe</h1>
<h1 *ngIf="nombre!='Pepe'">El nombre es distinto de Pepe</h1>

La sintaxis del *ngIf es la siguiente:

*ngIf="<condicion>"

La condicion va entre comillas dobles (""), si queremos especificar un string dentro de la condicion la podemos poner dentro de comillas simples ('')
