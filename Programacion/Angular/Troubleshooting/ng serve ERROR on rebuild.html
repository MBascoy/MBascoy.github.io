Al tener ejecutado ng serve cuando cambiamos muchos archivos de una vez (al cambiar de rama en el git por ejemplo) se produce un error al recompilar y se nos marcan errores que realmente no hay, simplemente con tirar el ng serve y volver a levantarlo se soluciona el problema

Solucion

Para evitar este problema ejecutamos ng serve de la siguiente manera:

ng serve --poll 2000
