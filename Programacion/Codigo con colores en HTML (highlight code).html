Descripcion

Metodo para tener trozos de codigo en paginas HTML formateado con colores segun el lenguaje usado

Metodo

Para poder introducir codigos con colores resaltando los diferentes elementos podemos usar paginas web como estas:

  • codebeautify.org
  • hilite.me
  • pinetools.com

Simplemente tenemos que introducir nuestro trozo de codigo e indicar el lenguaje y el estilo que queremos, la pagina nos generar√° un codigo HTML para embeber en cualquiera pagina.

Si queremos encontrar mas paginas de este estilo podemos googlear por: code highlight tool

Ejemplos

Lenguaje Java, Estilo: Rainbow

class HelloWorld
{
    // Tu programa comienza con una llamada a main().
    // Imprime "Hola Mundo" a la ventana de la terminal.
    public static void main(String args[])
    {
        System.out.println("Hola Mundo");
    }
}
