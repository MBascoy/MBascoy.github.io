Archivos binarios de proyecto

Mbascoy.dev / Ionic / Archivos binarios de proyecto
Descripcion

Localización de los archivos binarios que se generan al compilar un proyecto.

Si tenemos problemas con un proyecto, podemos borrar estos archivos para dejar el proyecto limpio, recompilar todo y partir desde ahí

Metodo

Las carpetas en las que están los archivos generados son:

  • .angular
  • node_modules
  • www
  • android/.gradle
  • android/capacitor-cordova-android-plugins
  • android/app/src/main/assets/public
  • android/app/build

Si borramos estos archivos/carpetas el proyecto quedará limpio y podremos recrear todo con los comandos:

npm install

Genera el node_modules

npx ionic build && npx cap copy

General el .angular, www y el android/app/src/main/assets/public

npx cap sync

General el android/capacitor-cordova-android-plugins

npx cap run android --target=emulator-5554 --external --no-sync

General el android/app/build y android/.gradle
