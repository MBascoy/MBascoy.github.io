Expresion TRUE || TRUE && FALSE

Explicacion

La expresion TRUE || TRUE && FALSE tiene dos posibles resultados dependiendo de si se lee de izquierda a derecha o de derecha a izquierda

Si la leemos de izquierda a derecha el resultado sera FALSE

Si la leemos de derecha a izquierda el resultado sera TRUE

Esto es importante ya que en lenguajes de programacion (como C o Java) la expresiones se analizan de derecha a izquierda siempre que los operadores tengan la misma prioridad.
