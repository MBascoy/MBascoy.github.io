Cambiar credenciales de acceso

Mbascoy.dev / Git / Cambiar credenciales de acceso
Descripcion

Como hacer para cambiar el usuario que usa git para conectarse al repositorio remoto.

Metodo Linux

Si tenemos configurado el credential.helper store las credenciales de acceso estaran guardadas en:

~/.git-credentials

Para que git nos permita acceder con otro usuario solo tenemos que borrar la linea que hace referencia al repositorio remoto dentro del archivo, una vez hecho esto al acceder al repositorio remoto git nos preguntara de nuevo por un usuario y contrasena.

Metodo Windows

Accedemos al Credential Manager desde el panel de control o desde el buscador de aplicaciones>

Entramos en la opcion de Windows Credentials y una vez dentro buscamos la entrada de git y la eliminamos, de esta manera la proxima vez que accedamos al repositorio remoto se nos pedira nuestro usuario y contrasena.
