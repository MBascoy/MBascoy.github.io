Configuracion inicial Git

Descripcion

Como realizar la configuracion inicial de Git despues de la instalacion y configurar un repositorio desde 0.

Metodo

Despues de instalar configuramos el usuario y el email:

> git config --global user.name "Manuel" > git config --global user.email "ejemplo@ejemplo.com"

NOTA: Esta informacion solo es para saber quien hace los commits, no es relativa a la cuenta de github ni nada parecido

Creamos una carpeta para nuestro proyecto e iniciamos el repositorio en esa carpeta

> mkdir proyecto > cd proyecto > git init

añadimos algun archivo al repositorio

> echo "hola que tal" > README.txt > git add README.txt

hacemos un commit al repositorio local

> git commit -m "añadido README"

ahora añadimos el repositorio remoto de github:

> git remote add origin https://github.com/MBascoy/repoprueba.git

y a continuacion hacemos un push del repositorio local al remoto:

> git push origin master

Se nos pedira el usuario y la contraseña de nuestra cuenta de github, en realidad no es la contraseña, si no el Personal Access Token (PAT) Si no lo tienes se puede regenerar desde la seccion settings de github

Con todo esto ya tenemos la informacion de nuestro repositorio local sincronizada con nuestro repositorio remoto
