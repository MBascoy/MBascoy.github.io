Comandos

Instalación WSL: https://learn.microsoft.com/en-us/windows/wsl/install

Instalar distro de ubuntu

wsl --install -d Ubuntu

Lanzar una imagen concreta

wsl -d Ubuntu

Listar imagenes de WSL:

wsl -l -v

Eliminar imagen (Distribucion) de WSL:

wsl --unregister Ubuntu

Establecer imagen por defecto:

wsl --set-default Ubuntu

Establecer version por defecto de WSL para las nuevas distribuciones instaladas:

wsl --set-default-version 2

Actualizar version de WSL para una distribución ya instalada

wsl --set-version Ubuntu 2

Parar la ejecución de una distribucion

wsl --shutdown

Listar distribuciones disponibles para descargar

wsl -l -o
