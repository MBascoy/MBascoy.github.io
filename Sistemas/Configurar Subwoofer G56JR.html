Configurar Subwoofer G56JR

Descripcion

Como configurar el subwoofer "SonicMaster" en el portatil Asus G56JR

Metodo Linux

Para realizar la configuracion del subwoofer en linux podemos hacerlo desde yast o editando los archivos de configuracion.

Usando Yast

Entramos en Yast y accedemos al menu de Sonido

Seleccionamos el dispositivo Xeon E3-1200 v3/4th Gen Core Processor HD Audio Controller en la lista y le damos a Editar. En teoria ese es el dispositivo de sonido asociado a los altavoces (y por lo tanto al subwoofer) el otro dispositivo creo que hace referencia a la salida de auriculares

Una vez dentro de la configuracion del dispositivo:

  • Seleccionamos Use the given board model. y pulsamos Editar
  • Se nos abre una nueva ventana donde tenemos que escribir una opcion, pondremos: asus-mode4
  • y pulsamos aceptar
  • Confirmamos que el valor se ha actualizado y pulsamos Siguiente
  • Pulsamos en Aceptar y el asistente de yast se encargara de guardar la configuracion y reiniciar los dispositivos de audio
  • Es importante que si tenemos alguna aplicacion abierta durante el proceso de guardado de la configuracion puede ser que perdamos el audio en esa aplicacion y tendremos que cerrarla y volver a abrirla


Modificando directamente los archivos de configuracion

Para realizar la configuracion del subwoofer directamente en los archivos de configuracion:

  • Abrimos el archivo /etc/modprobe.d/50-sound.conf:
    • # vim /etc/modprobe.d/50-sound.conf
  • E introducimos el siguiente codigo en el archivo (o realizamos las modificaciones necesarias para que quede igual
    • options snd-hda-intel model=,asus-mode4
    options snd slots=snd-hda-intel,snd-hda-intel
    # u1Nb.s9FVTfw97zC:8 Series/C220 Series Chipset High Definition Audio Controller
    alias snd-card-0 snd-hda-intel
    # 3hqH.bQjzHrSTHK8:Xeon E3-1200 v3/4th Gen Core Processor HD Audio Controller
    alias snd-card-1 snd-hda-intel

    La parte importante de la configuracion es options snd-hda-intel model=,asus-mode4 en este caso model=,asus-mode4 indica que se aplique ese modelo al segundo dispositivo de audio (la coma no es una errata, hace que se aplique esa configuracion solo al segundo dispositivo), si fuese model=asus-mode4,asus-mode4 se aplicaria a los dos, la clave esta en la coma.

  • Una vez editado el archivo guardamos y reiniciamos el sistema, el subwoofer ya deberia de funcionar despues de reiniciar
Metodo Windows

Para hacer que funcione el subwoofer en windows solo es necesario instalar el siguiente driver de Realtek:

Realtek_High_Definition_Audio_64bit_Win7_Win8_Win81_Win10_R279.exe

Una vez instalado reiniciamos el sistema y el subwoofer ya deberia de funcionar
