Comandos de Red

Mbascoy.dev / Linux / Comandos de Red
Descripcion

Configurar interfaz de red usando los nuevos comandos de configuración como ip(sustituto de ifconfig) y ss(sustituto de netstat)

Metodo

Configurar interfaz

Mostrar configuración de red:

> ip addr

Podemos simplificarlo como:

> ip a

Una interfaz puede tener asignada mas de una dirección IP, si solo queremos dejarle asignada una IP tenemos que añadir la nueva y borrar la antigua.

Añadir dirección de red, a una interfaz:

# ip addr add IP/MASCARA dev INTERFAZ

Ejemplo:

# ip addr add 192.168.1.56/24 dev eth0

Eliminar dirección de red de una interfaz:

# ip addr del IP/MASCARA dev INTERFAZ

Ejemplo:

# ip addr del 192.168.1.56/24 dev eth0

Tabla de rutas

Mostrar tabla de rutas:

> ip route

Modificar tabla de rutas para añadir ruta por defecto (gateway):

# ip route add default via IP_GATEWAY

Ejemplo:

# ip route add default via 192.168.1.1

Mostrar información conexiones

Mostrar solo sockets con conexión:

> ss -ntp

Mostrar solo sockets a la escucha:

> ss -ntpl

Mostrar ambos sockets a la escucha y con conexion:

> ss -ntpa
