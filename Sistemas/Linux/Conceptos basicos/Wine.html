Conceptos básicos de Wine

Establecer PREFIX

Para establecer un entorno virtual para el programa que vayamos a ejecutar lo hacemos estableciendo la variable de entorno WINEPREFIX

export WINEPREFIX=~/carpeta_entorno_wine

NOTA: Si se indica una carpeta en la que ya había un entorno de wine se usará con las configuraciones de dicho entorno, si es una carpeta vacía se creará un entorno desde 0.

Configurar Wine

Para configurar el entorno una vez establecamos el WINEPREFIX lo hacemos con:

winecfg
Lanzar aplicaciones con Proton

Si queremos lanzar alguna aplicación con proton (principalmente juegos), lo hacemos con:

protontricks-launch aplicacion.exe

Una vez ejecutemos este comando se nos preguntará que entorno queremos usar, aparecerá una lista de entornos con todos los entornos que se hayan creado en steam referentes a cada uno de los juegos, seleccionaremos el entorno que tenga la versión de proton que queramos para ejecutar nuestra aplicación.
