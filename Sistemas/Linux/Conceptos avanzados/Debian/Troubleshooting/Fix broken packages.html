Fix broken packages

Mbascoy.dev / Linux / Debian / Fix broken packages
Metodo

Si tenemos paquetes que tienen depedencias pendientes (broken packages) podemos arreglarlos con el siguiente comando:

# apt --fix-broken install

Este comando intentará buscar e instalar las dependencias de los paquetes en los repositorios que tengamos configurados.
