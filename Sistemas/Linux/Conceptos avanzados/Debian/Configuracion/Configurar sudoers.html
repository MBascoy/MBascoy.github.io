Descripcion

Configurar un usuario para que pueda utilizar el comando sudo

NOTA: al usar el comando sudo en Debian se usa el path con las rutas "sbin" por eso hay comandos que no se detectan como instalados al usar un usuario normal, incluso cuando usamos el comando su para convertirnos en root en Debian no se a침aden las rutas "sbin", podriamos a침adirlas manualmente, pero lo suyo es hacer uso del comando sudo

Metodo

Cuando un usuario no est치 configurado para utilizar el comando sudo obtenemos el siguiente error al intentar utilizarlo:

MiUsuario is not in the sudoers file. This incident will be reported

Para configurar nuestro usuario para que pueda utilizar sudo solo tenemos que a침adirlo al grupo correcto del sistema con el siguiente comando:

# usermod -aG sudo MiUsuario

NOTA: en los sistemas basados en debian el grupo es sudo y en sistemas basados en redhat el grupo es wheel

Una vez hecho esto reiniciamos el sistema y ya podremos usar sudo con nuestro usuario.
