Usar logger Slf4j

Descripcion

Como utilizar Slf4j para logear mensajes de error o informativos

Solucion

Dependencias

Al estar utilizando Spring solo tenemos que indicar la dependencia del lombok en el archivo pom.xml:

<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.20</version>
    <scope>provided</scope>
</dependency>
Metodo

Para utilizar el Slf4j primero tenemos que indicar la siguiente anotacion a nivel de la clase donde lo queramos utilizar:

@Slf4j

Y a continuaci√≥n solo tenemos que utilizar el objeto log para generar mensajes utilizando los distintos metodos disponibles:

  • info
  • error
  • warn
  • debug

Ejemplo:

log.info("Calling to getAllMembersData()");

NOTA

Este metodo es para utilizar el logger usando spring, si queremos usarlo en un programa java normal es necesario realizar algunas implementaciones a mayores (Pendiente de documentar)
