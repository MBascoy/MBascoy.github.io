Consultar version Spring y Spring Boot

Descripcion

Como saber que version de Spring y Sprinb Boot estamos usando en nuestro proyecto

Metodo

La versión de Spring Boot aparece cuando ejecutamos el proyecto (mvn spring-boot:run), justo debajo del logo de Spring:

Para saber la versión de Spring Framework tenemos que incluir la siguiente linea en nuestro código:

System.out.println("Spring Version:" + SpringVersion.getVersion());

y su import:

import org.springframework.core.SpringVersion;

Ahora al ejecutar nuestra aplicación de Spring veremos la versión:
