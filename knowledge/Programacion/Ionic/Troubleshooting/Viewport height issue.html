Viewport height issue

Descripcion

Creando una aplicación de ionic completamente desde 0, el template inicial no carga bien la página inicial, debido a que no es capaz de establecer bien el alto del viewport.

Las dos tabs tienen exactamente el mismo código, pero se puede ver una diferencia en el centrado del texto:

El problema siempre se reproduce en la página que carga primero, si nos ponemos en la tab2 y hacemos actualizamos la página el problema se dará al reves...

El fallo se "resuelve" si redimensionamos la ventana del navegador (desde el pc) o si cambiamos la orientación de portrait a landscape y volvemos otra vez a portrait (desde un dispositivo movil).

Solucion

Para solucionar este problema de manera permanente podemos introducir el siguiente código en el archivo global.scss:

html.body {
    height: 100vh;
    width: 100%;
    padding: 0px;
    margin: 0px;
}
