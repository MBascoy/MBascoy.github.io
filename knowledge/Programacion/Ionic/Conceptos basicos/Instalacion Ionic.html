Instalacion de Ionic.

Página de Ionic Framework

Metodo

Primero tenemos que tener instaldo Node, podemos ver como instalarlo aqui: Instalar node con nvm

Para instalar el cli de Ionic para poder generar proyecto lo hacemos con el siguiente comando:

npm install -g @ionic/cli

Una vez tenemos instalado el cli podemos generar un nuevo proyecto con el siguiente comando:

ionic start

Al lanzar ese comando se nos preguntará de manera interactiva por nombre de la alicación, framework a usar (Angular, react o vue) y algunas opciones de configuración

Si queremos podemos lanzar el siguiente comando para generar un proyecto de Ionic con Angular y con un template de pestañas (tabs):

ionic start mi_aplicacion --type=angular tabs

Una vez tenemos creada la aplicación entramos en la carpeta y la ejecutamos con el siguiente comando:

cd mi_aplicacion ionic serve
Instalación IOS

Capacitor IOS

Para poder ejecutar una App de Ionic en Mac con el emulador de IOS necesitamos tener XCode instalado y ejecutar los siguientes comandos:

xcode-select --install brew install cocoapods
