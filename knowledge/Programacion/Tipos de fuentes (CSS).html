Tipos de fuentes CSS

Explicacion

Para aplicar el tipo de fuente usamos el atributo Font-Family.

Hay principalmente 5 tipos de fuentes genericas en CSS:

  • Serif
  • Sans-serif
  • Monospacec
  • Cursive
  • Fantasy

Dentro de cada tipo de fuente generico podemos tener varios tipos distintos de fuentes, pero todos ellos siguen el patron de diseno del tipo generico:

Ejemplo

Al indicar un tipo de fuente podemos indicar varios separados por comas, asi si un navegador no soporta un tipo concreto intentara utilizar otro de la lista


font-family: "Lucida Console", "Courier New", monospace;
