Arrays asociativos

Mbascoy.dev / JavaScript - TypeScript / Arrays asociativos
Descripcion

Los arrays asociativos nos permiten usar como identificador de cada elemento del array un string en vez de un numero, la manera de usarlos es la siguiente:

var elements = {first:'value1', car:'ferrari'};

Manera alternativa:

var elements = {"first":'value1', "car":'ferrari'};

Para acceder a cada elemento se haría tal que así:

elements["car"]

El siguiente código:

var elements = {first:'value1', car:'ferrari'};

console.log(elements["first"]);
