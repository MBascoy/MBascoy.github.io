Array eliminar varios elementos

Descripcion

Como eliminar varios elementos de un array usando un sub-array

Metodo
const removeItems = (array, itemsToRemove) => { 
  return array.filter(v => { 
    	return !itemsToRemove.includes(v); 
    });
} 
removeItems(["gato", "perro", "oveja"], ["gato", "oveja"]);

Este código crea una función removeItems que podemos llamar pasandole un array y los elementos que queremos eliminar del mismo.

En este caso concreto se eliminarían "gato" y "oveja" del array, dejando solo "perro"
