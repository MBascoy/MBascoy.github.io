Manejar eventos con AddEventListener

Mbascoy.dev / JavaScript - TypeScript / Manejar eventos con AddEventListener
Descripcion

Podemos usar addEventListener para manejar eventos al interactuar con elementos de la página, de igual manera que usamos funciones como onClick() en una etiqueta para asociar el evento a una función.

Metodo

Para utilizar el addEventListener y asignar un evento de un elemento a una función lo hacemos con la siguiente sintaxis:

elemento.addEventListener(evento, funcion)

Lista de Eventos

Un ejemplo es el siguiente, asignamos el evento click a un elemento boton y hacemos que llame a la función mensaje:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
    </head>
    <body>
        <button id="boton">BOTON</button>

        <script>
            let boton = document.getElementById("boton");

            function mensaje(){
                alert("hola")
            }

            boton.addEventListener("click", mensaje)
        </script>
    </body>
</html>

preventDefault

Podemos usar esta propiedad para evitar que el elemento que tiene asociado el evento realice la función que normalmente realiza por defecto.

Por ejemplo por defecto un checkbox al pulsarlo se queda marcado, con está propiedad evitaríamos que se produzca eso y simplemente podemos manejar el eventos del "click"
