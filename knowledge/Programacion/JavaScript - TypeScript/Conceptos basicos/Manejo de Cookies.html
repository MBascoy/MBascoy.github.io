Manejo de Cookies

Descripcion

Como manejar las cookies del navegador con JavaScript.

Metodo

Para acceder a las cookies utilizamos la siguiente linea de código:

document.cookie

Ejemplo para guardar, leer y borrar una cookie:

<html>
    <body>
        <button onclick="guardarCookie()">Guardar Cookie</button>
        <button onclick="verCookie()">Ver Cookie</button>
        <button onclick="borrarCookie()">Borrar Cookie</button>

        <script>

            function guardarCookie() {
                document.cookie = "nombre=Pepe; max-age=3600";
            }

            function verCookie() {
                alert(document.cookie)
            }

            function borrarCookie() {
                document.cookie = "nombre=; max-age=0";
            }
        </script>
    </body>
</html>

Para leer simplemente el valor de una cookie concreta podemos usar el siguiente código:

const cookieValue = document.cookie
    .split('; ')
    .find((row) => row.startsWith('nombre='))
    ?.split('=')[1];
alert(cookieValue)

Tenemos que modificar nombre= por el nombre de la cookie que queramos consultar.
