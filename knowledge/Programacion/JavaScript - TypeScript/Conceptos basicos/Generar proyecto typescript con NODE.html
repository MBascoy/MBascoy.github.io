Generar proyecto typescript con NODE

Descripcion

Como generar un proyecto Typescript usando node y usar el transpilador tsc para convertir el codigo a javascript.

Metodo

https://khalilstemmler.com/blogs/typescript/node-starter-project/

mkdir typescript-starter npm init -y npm install typescript --save-dev npm install @types/node --save-dev npx tsc --init --rootDir src --outDir build \ --esModuleInterop --resolveJsonModule --lib es6 \ --module commonjs --allowJs true --noImplicitAny true mkdir src npx tsc
