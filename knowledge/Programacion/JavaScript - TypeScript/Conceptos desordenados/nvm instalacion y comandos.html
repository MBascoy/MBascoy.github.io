Metodo
  • Instalamos nvm con el siguiente comando (version 0.39.4):
    • curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.4/install.sh | bash
  • Instalar una version de node:
    • nvm install 16.14
  • Instalar la ultima version disponible de node:
    • nvm install node
  • Listar las versiones de node instaladas con nvm:
    • nvm ls
  • Establecer una version de node por defecto:
    • nvm alias default 16.14
  • Cambiar la version de node temporalmente (para la sesion de la terminal que tengamo activa):
    • nvm use 16.14
