Emulador no inicia Chrome

Descripcion

Al intentar ejecutar Google Chrome en un dispositivo emulado se queda cargando eternamente.

Solución

Para solucionar este problema creamos un archivo advancedFeatures.ini dentro de la carpeta .android que hay en nuestra carpeta persona, la ruta sería %USERPROFILE%\.android\

El contenido del archivo será el siguiente:

Vulkan = off
GLDirectMem = on

Una vez hecho esto el navegador ya se debería de cargar de forma correcta.
