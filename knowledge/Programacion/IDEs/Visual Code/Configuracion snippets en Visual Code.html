Configuracion snippets en Visual Code

Descripcion

En que archivo y que formato hay que seguir para configurar code snippets en el editor Visual Code.

Para usar los snippets en Visual Code solo tenemos que introducir el comando del snippet y pulsar la tecla TAB

Configuracion

En Visual Code los snippets están configurados en varios archivos en la carpeta %USERPROFILE%\AppData\Roaming\Code\User\snippets, un archivo por cada lenguaje.

Por ejemplo, los snippets de java van en el archivo java.json, los de typescript en typescript.json

Si queremos tambien podemos acceder directamente a los snippets desde el propio IDE desde File > Preferences > User Snippets y a continuación seleccionar el lenguaje.

La estructura de los snippets es la siguiente:

"For Loop": {
    "prefix": ["sop"],
    "body": ["System.out.println(\"${1:Text}\");"],
    "description": "A for loop."
}
