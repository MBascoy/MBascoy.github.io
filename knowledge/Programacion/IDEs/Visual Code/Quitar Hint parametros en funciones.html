Quitar Hint parametros en funciones

Mbascoy.dev / IDEs / Quitar Hint parametros en funciones
Descripción

Cuando escribimos una función con parámetros vemos el siguiente Hint antes de los parámetros:

Podemos ver el nombre del parámetro antes del valor del propio parámetro.

Metodo

Para quitar esto nos vamos a File > Preferences > Settings

Buscamos por inlay y marcamos la opción como off

Una vez hecho esto ya no aparecerá el texto antes de los parámetros.
