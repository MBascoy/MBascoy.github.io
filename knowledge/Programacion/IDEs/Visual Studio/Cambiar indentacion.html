Cambiar indentacion

Descripcion

Como cambiar el tamaño de indentación en un archivo entero usando Visual Studio.

Metodo
  • Primero nos vamos a: Tools > Options...
  • En la ventana de options seleccionamos Text Editor
  • uscamos el lenguaje en el que estemos programado, C# en este caso, y seleccionamos la subsección tabs
  • Seleccionamos la indentación en espacios y ponemos el numero de espacios con el que queremos indentar.
  • Ahora nos vamos a: Edit > Advanced > Format Document

Despues de esto nuestro documento ya estará indentado de manera automática con el número de espacios que le hallamos indicado.

Cuando veamos esta indicación tenemos dos opciones para dejar solo espacios o solo tabuladores:

  • Hacemos click encima de MIXED e indicamos si queremos espacios o tabuladores.
