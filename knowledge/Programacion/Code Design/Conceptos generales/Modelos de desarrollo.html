Descripción

Un modelo de desarrollo indica como se llevan a cabo las distintas fases de ciclo de vida del desarrollo de un software (SDLC) (Software Development Life Cicle)

Modelos

Existen varios tipos de modelos:

  • Modelo informal
  • Waterfall
  • Modelo en V
  • Iterativo
  • Spiral
  • Incremental
Elección de modelo

Si es un proyecto pequeño podemos elegir el modelo Waterfall

Para un proyecto mediano podemos elegir un modelo Iterativo, Spiral o Incremental

Para un proyecto grande es mejor usar modelos secuenciales como Waterfall o el Modelo en V
