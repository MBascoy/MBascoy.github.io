Naming Convention

Java

Java Naming Conventions

Tipo Identificador Norma Ejemplo
Package Todas las letras en minuscula y los diferentes componentes del paquete separados por puntos

com.example.main

com.sun.eng

com.apple.quicktime.v2
Classes PascalCase, la primera letra de cada palabra en mayuscula

class PepeClass

class ImageSprite;
Interfaces PascalCase, la primera letra de cada palabra en mayuscula

interface RasterDelegate;

interface Storing;
Methods Deben ser verbos, camelCase, la primera letra de la primera palabra en minuscula y la primera letra de las siguientes palabras en mayuscula

getBackground();

run();

addEmployee();
Variables camelCase, la primera letra de la primera palabra en minuscula y la primera letra de las siguientes palabras en mayuscula

int stockQuantity;

String name;

float weight;
Constants Uppercase separando las palabras por underscore "_", todas las letras de una constante van en mayuscula

static final float NUMERO_PI = 3.14

static final int PESO_MAXIMO = 300

static final int DIAS_SEMANA = 7
