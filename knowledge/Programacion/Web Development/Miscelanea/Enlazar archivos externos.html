Enlazar archivos externos

Descripcion

Como enlazar archivos externos a un archivo HTML como un archivo javascript o css.

Enlazar archivo JavaScript

Para enlazar codigo javascript en un archivo externo lo haremos de la siguiente manera:

<script src="myScript.js"></script>

Para hacer que el JavaScript se cargue despues del DOM (o sea al final del todo), usamos el atributo defer asi:

<script src="myScript.js" defer></script>
Enlazar archivo CSS

Para enlazar codigo CSS en un archivo externo lo haremos de la siguiente manera:

<link rel="stylesheet" href="mystyle.css">
