Descripcion

Como utilizar la API de twitter para obtener el feed de una cuenta.

Metodo

Para obtener el feed con el bearer token lo hacemos con la siguiente solicitud CURL:

curl -X GET 'https://api.twitter.com/2/users/ID_CUENTA_TWTTER/tweets?tweet.fields=created_at,text,attachments,source' \
-H "Authorization: Bearer MI_BEARER_TOKEN

Cambiamos ID_CUENTA_TWITTER por el id de la cuenta de la que queremos obtener el feed (el id podemos sacarlo con una web como esta: https://twiteridfinder.com/)

Y cambiamos MI_BEARER_TOKEN por el bearer token de la cuenta de desarrollador.
