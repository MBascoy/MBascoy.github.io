Etiquetas basicas

Mbascoy.dev / Web Development / Etiquetas basicas
Descripcion
  • <div>: se utiliza para ordenar bloques de codigo o añadir un mismo estilo a un bloque de codigo
  • <span>: se utiliza para dar estilo a un pequeño trozo de texto
  • <ul>: lista desordenada
  • <ol>: lista ordenada
  • <video>: Introducir un video en la página, al igual que <img> utiliza el atributo src <video src="video/video.mp4"></video>

    • Podemos indicarle el atributo controls para que muestre los controles del video e incluso incluir un texto entre las etiquetas para que se muestre ese texto en caso de que no se pueda cargar el video

    <video src="video/video.mp4" controls>El video no se puede mostrar</video>
Alternativas a Div

En vez de utilizar etiquetas div para organizar todo (se conoce como divitis) html5 provee de etiquetas con un sentido mas semántico para organizar bloques de código.

Entre estas etiquetas podemos encontrar:

  • header: Para la cabecera de la pagina, la parte superior. NO confundir con la etiqueta head
  • nav: para el menu de navegacion
  • main: para el cuerpo principal de la página
  • section: para una seccion concreta
  • footer: para el pie de la página, la parte inferior.
