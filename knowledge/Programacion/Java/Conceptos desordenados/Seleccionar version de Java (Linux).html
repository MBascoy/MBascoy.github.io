Seleccionar version de Java (Linux)

Descripcion

Como seleccionar el JDK/JRE que se utilizará por defecto si hay varias versiones instaladas.

Metodo para OpenSUSE

Metodo

Para consultar todos los JDK configurados ejecutamos el siguiente comando (como root):

# update-alternatives --list java

Para configurar que versión queremos utilizar usamos el siguiente comando:

# update-alternatives --config java

Se nos mostrará un menu como el siguiente en el que se indica cual es el JDK actualmente configurado y nos dará la opción de inidcar otro para usar por defecto:
