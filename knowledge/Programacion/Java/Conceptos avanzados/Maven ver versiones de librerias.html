Maven ver versiones de librerias

Descripcion

Como ver las versiones y las dependencias de las librerías que estamos usando en nuestro proyecto.

Consultar versiones

Para consultar todas las librerías que se están usando en un proyecto, su versión, y sus dependencías, usamos el siguiente comando:

mvn dependency:tree

Tambien podemos usar el siguiente comando que nos muestra el pom definitivo que se usará incluyendo las versiones de las librerías:

mvn help:effective-pom

La salida de ese comando a veces puede ser muy extensa, podemos hacer un pipe con less:

mvn help:effective-pom | less
