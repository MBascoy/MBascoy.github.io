Error dependencia jasmine

Version
  • Nodejs: 16.5.0
  • npm: 7.19.1
  • AngularCLI: 10.0.8
  • Angular: 10.0.14
Descripcion

Al crear un nuevo proyecto obtenemos el siguiente error:

npm ERR! ERESOLVE unable to resolve dependency tree
npm ERR! 
npm ERR! While resolving: mi-app@0.0.0
npm ERR! Found: jasmine-core@3.5.0
npm ERR! node_modules/jasmine-core
npm ERR!   dev jasmine-core@"~3.5.0" from the root project
npm ERR! 
npm ERR! Could not resolve dependency:
npm ERR! peer jasmine-core@">=3.8" from karma-jasmine-html-reporter@1.7.0
npm ERR! node_modules/karma-jasmine-html-reporter
npm ERR!   dev karma-jasmine-html-reporter@"^1.5.0" from the root project
Resolucion

Para resolver el error simplemente entramos en el directorio de la aplicacion que se ha creado y ejecutamos el siguiente comando:

> cd mi-app > npm install --legacy-peer-deps

Una vez ejecutado esto podemos ejecutar ng serve y ya podremos accedera a nuestro proyecto con http://127.0.0.1:4200
