Array - Cannot read properties of undefined

Descripcion

Al declarar un array y utilizar el metodo push vemos el siguiente error:

ERROR TypeError: Cannot read properties of undefined (reading 'push')
Solucion

El error se produce porque al declarar el array no se inicializa.

En este caso el array esta declarado INCORRECTAMENTE asi:

personas : Array<string>;

La manera CORRECTA ser√≠a la siguiente;

personas : Array<string> = [];
Declarar un array en Angular/Javascript

Podemos declarar un array de dos maneras:

personas : Array<string> = []; personas : string[] = [];
