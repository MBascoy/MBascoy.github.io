File Upload

Descripcion

Como enviar archivos desde frontend para que se almacenen en un backend.

En este ejemplo se envían dos datos, un nombre y un archivo, despues el back se encarga de guardar el archivo y a mayores el nombre que le pasemos y el nombre del archivo subido en la base de datos.

La parte de BackEnd la podemos consultar AQUI

Metodo

El código podemos descargarlo en este repositorio.

Para seleccionar el archivo usamos un input de tipo file

Tenemos que tener en cuenta que para subir un archivo a la vez que otros datos tenemos que utilizar un form-data.
