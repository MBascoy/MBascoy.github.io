Comando Prettier

Descripcion

Como instalar el comando prettier para poder formatear varios archivos

Documentación prettier

Metodo

Para instalar el comando lo hacemos asi:

npm install prettier

Una vez instalado podemos indicarle formatear todos los archivos del proyecto tal que así:

prettier --write .

Si queremos ser mas especificos con que archivos filtrar tenemos varias opciones

El siguiente comando formatea todos los archivos .js del proyecto

prettier --write "**/*.js"

Si queremos podemos formatear los archivos de una sola carpeta:

prettier --write src/features/clients/

Para formatear todos los archivos .css, .js y .ts dentro de la carpeta src y todas las carpetas anidadas:

prettier --write "src/**/*.{css,js,ts}"
