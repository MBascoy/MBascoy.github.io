Heap out of memory

Descripcion

Al ejecutar un proyecto podemos obtener el siguiente error:

FATAL ERROR: Reached heap limit Allocation failed - JavaScript heap out of memory
Solucion

Para solucionar el error antes de ejecutar el proyecto (ng serve or npm run start) tenemos que establecer la siguiente variable de entorno:

export NODE_OPTIONS="--max-old-space-size=8192"
