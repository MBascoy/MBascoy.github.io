Como crear un nuevo proyecto angular usando el Angular-CLI

Para generar un nuevo proyecto utilizamos el siguiente comando:

> ng new <Nombre Proyecto>

Con este comando se nos creará toda la estructura de ficheros del proyecto en un directorio con el nombre de proyecto que le hemos indicado

Una vez creado el proyecto podemos iniciarlo con el siguiente comando

ng serve

Este comando nos compilará el proyecto y levantará un servidor web para hacerlo accesible a traves del puerto 4200 por defecto, de manera que podemos ver nuestro proyecto accediendo a la direccion http://127.0.0.1:4200
