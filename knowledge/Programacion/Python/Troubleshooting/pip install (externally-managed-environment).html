pip install (externally-managed-environment)

Descripcion

Al intentar instalar algo con pip obtenemos el siguiente error:

error: externally-managed-environment
Metodo

El error se da porque no estamos lanzando pip desde un entorno virtual de python (venv)

Si queremos poder lanzarlo de manera global tenemos que crear el siguiente archivo:

~/.config/pip/pip.conf

Y añadimos el siguiente contenido:

[global]
break-system-packages = true

Una vez hecho esto nos dejará ejecutar pip sin necesidad de estar dentro de un entorno virtual.
