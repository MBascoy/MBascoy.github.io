Descripcion

Como declarar funciones en Python.

Metodo

La sintaxis para declarar funciones en python es la siguiente:

def nombre_funcion(param1, param2):
    return valor

Por ejemplo:

def sumar(numA, numB):
    return numA + numB

print(sumar(5,8))

Cuando llamamos a una funcion podemos simplemente indicar el valor del parámetro o tambien indicar el nombre del parámetro, en el ejemplo anterior podemos llamar a sumar asi:

sumar(numA=5, numB=8)
