Descripcion

Metodo

Para crear diccionarios lo hacemos de la siguiente manera:

mi_dict = {"animal": "gato", "objeto": "libro"}

Para acceder a un valor del diccionario lo hacemos usando la clave del valor que queremos acceder:

mi_dict["animal"] # devuelve "gato"

Si solo quisiesemos obtener la lista de claves o la lista de valores podemos hacerlo de la siguiente manera:

mi_dict.keys() mi_dict.values()

Devuelve: dict_values(['gato', 'libro'])

Estos dos metodos devuelven un tipo dict_values que no puede ser iterado, si queremos hacerlo mas manejable podemos convertirlo a una lista de la siguiente manera:

list(mi_dict.keys()) list(mi_dict.values())
