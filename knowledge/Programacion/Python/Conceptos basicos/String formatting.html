String formatting

Descripcion

Como utilizar el formateado de texto en python.

Metodo

Para usar el string formatting lo hacemos de la siguiente manera:

"texto %s" % variable

En el string se utilizar√°n indicadores de formato para que sean substituidos por las variables que se indican despues del % que va despues del string

Los indicadores de formato son los siguientes:

  • %c: Formato para un caracter
  • %s: Formato para un string
  • %d: Formato para un entero
  • %f: formato para un numero flotante

Si tenemos varias variables para ser substituidas en el string tenemos que indicarlas despues del % poniendolas entre parentesis () y separadas por comas (,) en el orden en el que aparecen en el string

Ejemplos

mi_numero = 7
mi_string = "ejemplo"
"mi numero es: %d" % mi_numero
"mi numero es: %d y mi string es: %s" % (mi_numero, mi_string)

El primer string devuelve: "mi numero es: 7"

El segundo string devuelve: "mi numero es: 7 y mi string es: ejemplo"
