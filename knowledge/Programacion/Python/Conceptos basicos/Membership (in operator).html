Membership (in operator)

Mbascoy.dev / Python / Membership (in operator).html
Descripcion

Podemos utilizar el operador in para comprobar en varios tipos de datos (strings, listas, tuplas...) si contienen un valor.

Metodo

El operador in lo usamos de la siguiente manera:

  • Comprobar substring en una cadena:
    • 'pe' in 'pera' #devuelve True
  • Comprobar que una lista contiene un valor:
    • 16 in [1,5,4,16,4,8] # devuelve True
  • Comprobar que una lista contiene un string:
    • "a" in ["hola", "adios"] # devuelve False

    El ejemplo anterior devuelve false a pesar de que "hola" contiene una "a", ya que se busca la comparación de los elementos de la lista y no de los elementos del string interno

    Si quisiesemos buscar en el string interno tendríamos que indicarlo de esta manera (el 0 del final hace referencia al primer elemento de la lista):

    "a" in ["hola", "adios"][0] # devuelve True "hola" in ["hola", "adios"] # devuelve True
  • Comprobar si un diccionario contiene un valor:
    • "animal" in {"fruta": "manzana", "animal": "gato"} # devuelve True "gato" in {"fruta": "manzana", "animal": "gato"} # devuelve False

    Por defecto la busqueda se hace en la key del diccionario

  • Comprobar si un diccionario contiene un valor en el value
    • "gato" in {"fruta": "manzana", "animal": "gato"}.values() # devuelve True
