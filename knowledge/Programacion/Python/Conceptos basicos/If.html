If

Descripcion

Como utilizar if statement en python.

Metodo

La sintaxis es la siguiente:

if condicion:
  codigo
elif condicion:
  codigo
else:
  codigo

Por ejemplo:

if color=="rojo":
    printRed()
elif color=="verde":
    printGreen()
else:
    printOther()

Podemos encadenar varias condiciones con and o or

color="yellow"
forma="triangulo"

if color=="red":
    print("rojo")
elif color=="verde" or forma=="triangulo":
    print("verde")

Pass

Una sentencia que tiene python es pass que simplemente no hace nada, pero puede ser util para que el c√≥digo sea mas legible en ciertos casos:

if color=="rojo":
    printRed()
elif color=="verde":
    printGreen()
else:
    pass

De esta manera dejamos mas claro que en el caso de else no hacemos nada.
