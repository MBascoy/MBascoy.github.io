Factory Functions

Mbascoy.dev / Python / Factory Functions.html
Descripcion

Son funciones que nos permiten generar tipos de datos en base a datos de otro tipo.

Metodo
  • Generar un int
    • int("4")
  • Generar un string
    • str(4)
  • Generar una lista
    • list(4,5,6)
  • Generar una tupla
    • tuple(4,5,6)
  • Generar un diccionario
    • dict(animal="gato", objeto="libro")

    NOTA: Al generar un diccionario con la function dict() la clave se indica sin comillas ""
