Descripcion

Las listas en python nos permiten almacenar varios valores y se accedidos mediante un indice.

Metodo

Para crear una lista lo hacemos de la siguiente manera:

my_list = [1,2,3,4]

Las listas pueden contener cualquier tipo de dato, incluso varios tipos de datos dentro de la misma lista:

my_list = ["uno", 2, "coche"]

Las listas tambien pueden contener otra lista anidada:

my_list = ["manzana", [1,2,3], "pera"]
