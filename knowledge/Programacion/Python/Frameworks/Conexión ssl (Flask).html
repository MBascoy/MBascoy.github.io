Conexión ssl (Flask)

Descripcion

Como hacer para que flask use una conexión segura (ssl).

Metodo

Para hacer que flask establezca conexiónes cifradas lo hacemos cambiando el app.run de la siguiente manera:

app.run(host="0.0.0.0", ssl_context='adhoc', debug=True)

Solo tenemos que añadir la parte de ssl_context='adhoc' y las conexiones se harán de manera cifrada.

La opción host="0.0.0.0" no tiene nada que ver en este caso, pero se utiliza para que se acepten las peticiones de cualquiera interfaz, si no indicamos esta opción solo se aceptan las conexiones locales (localhost)
