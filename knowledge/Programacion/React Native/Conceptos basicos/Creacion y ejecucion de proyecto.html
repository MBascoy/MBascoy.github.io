Creacion y ejecucion de proyecto

Descripcion

Como crear un proyecto base con React Native, y comandos a usar para ejecutarlo.

Metodo

Creacion

Para crear una aplicación con React Native, lanzamos el siguiente comando desde la consola (Nos creará una carpeta con toda la estructura de nuestro proyecto):

npx react-native init MiPrimeraApp

Ejecucion

A continuación para ejecutar la aplicación primero necesitamos ejecutar Metro que se encargará de empaquetar nuestra aplicación, lo hacemos lanzando el siguiente comando:

npx react-native start

NOTA: Tenemos que dejar esta consola abierta con el comando en ejecución.

NOTA2: Este comando lo tenemos que ejecutar desde la carpeta donde tengamos nuestros proyecto en este caso la carpeta de MiPrimeraApp.

A continuación ejecutamos el siguiente comando para lanzar la App (Desde la carpeta del proyecto), tendremos que abrir una nueva consola para lanzar este comando, no tenemos que cerrar la consola donde estámos ejecutando Metro.

npx react-native run-android
