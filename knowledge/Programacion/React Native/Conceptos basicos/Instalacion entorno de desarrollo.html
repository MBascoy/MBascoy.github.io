Instalacion entorno de desarrollo

Descripcion

Instalar todos los elementos necesarios para poder desarrollar con React Native.

Metodo

Usaremos Android Studio y el SDK 31 que es el de android 12 (S) para desarrollar aplicaciones con React Native.

Pasos:

  • Descargamos el instalador de Android Studio (Descarga)
  • En la instalación seleccionamos los siguientes componentes:
    • Android SDK
    • Android SDK Platform
    • Android Virtual Device
    • Performance (Intel ® HAXM)
  • Una vez instalado en la configuración inicial instalamamos los siguientes paquetes desde el SDK manager:
    • Android SDK Platform 31
    • Intel x86 Atom_64 System Image

A continuación tenemos que configurar las variables de entornos:

  • ANDROID_HOME
  • PATH
