Switch

Descripcion

Como utilizar la sentencia switch en C#.

Metodo

El switch podemos usarlo de la siguiente manera (de la misma manera que en otros lenguajes):

String nombre = "gato";
String tipo;

switch (nombre)
{
    case "gato":
        tipo = "animal";
        break;
    case "cactus":
        tipo = "planta";
        break;
    case "coche":
        tipo = "objeti";
        break;
    default:
        tipo = "desconocido";
        break;
}

WriteLine(tipo);

En C# contamos con una estructura del switch mucho mas limpia, el código anterior se podría escribir de la siguiente manera:

String nombre = "gato";
String tipo;

tipo = nombre switch
{
    "gato" => "animal",
    "cactus" => "planta",
    "coche" => "objeto",
    _ => "desconocido"
};

WriteLine(tipo);
