String interpolation y Verbatim string

Descripcion

  • String interpolation:

    • Nos permite integrar varibles directamente dentro de un String

  • Verbatim String:

    • Nos permite componer un string de manera que las secuencias de escape no se procesen y se muestren como parte de string

String interpolation

Para utilizar el string interpolation incluimos el simbolo $ antes del string y a continuacion podemos introducir variables dentro del string rodeadas por {}, por ejemplo:

string ejemplo = "hola";
Console.Write($"valor de variable: {ejemplo}");
Verbatim string

Para utilizar el Verbatim string incluimos el simbolo @ antes del string, asi el string que pongamos a continuacion no procesara las secuencias de escape, por ejemplo:

Console.Write(@"Frase \n de \n ejemplo");
