If

Descripcion

Como utilizar la sentencia if en C#.

Metodo

En C# usamos el if de la siguiente manera:

if (numero > 5)
{
    WriteLine("Es mayor de 5");
}
else
{
    WriteLine("Es menor o igual a 5");
}

Pattern matching

Una funcionalidad del if en C# es el pattern matching, que nos permite usar la palabra reservada "is" para hacer comparaciones de tipos.

object numero = "8";

if (numero is int i)
{
    WriteLine($"Es un entero: {i}");
}
else
{
    WriteLine("No es un entero");
}

En el ejemplo anterior la salida sería "No es un entero".

Pero si en la declaración de numero le quitamos las comillas la salida sería, "Es un entero: 8".
