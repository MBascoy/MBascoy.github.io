For y foreach

Descripcion

Como utilizar la sentencia for y foreach en C#.

Metodo

Usamos el for de la siguiente manera:

for (int i = 0; i < 10; i++)
{
    WriteLine(i);
}

foreach

El foreach podemos usarlo sobre colecciones de tipos de datos, como por ejemplo un array:

string[] nombres = new[] {"Pepe","Maria","Paco"};

foreach (string nombre in nombres)
{
    WriteLine($"{nombre} tiene una longitud de {nombre.Length}");
}

Para poder usar foreach sobre un tipo de datos tiene que cumplir lo siguiente:

  • El tipo debe de tener un metodo GetEnumerator que devuelva un objeto
  • El objeto devuelto debe de tener una propiedad Current y un metodo MoveNext
  • El metodo MoveNext debe cambiar el valor de Current y devolver true si hay mas elemento sobre los que iterar y false si no hay mas elementos.
