LocalDB manejar conexiones

Descripcion

Como manejar instancias de LocalDB y conectarse para hacer consultas.

Metodo

Consultar instancias de LocalDB

Para consultar las instancias creadas de LocalDB lo hacemos con el siguiente comando:

SqlLocalDB info

Para consultar información de una instancia en concreto lo hacemos asi:

SqlLocalDB info "instancia"

Si tenemos una instancia que está en estado "Stopped" podemos arrancarla de nuevo con el siguiente comando:

SqlLocalDB start "instancia"

Conectarnos a una instancia

Una vez tenemos arrancada una instancia de localdb podemos conectarnos a ella usando SQL Server Management Studio(SSMS).
