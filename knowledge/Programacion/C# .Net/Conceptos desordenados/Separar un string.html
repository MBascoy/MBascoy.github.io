Separar un string

Descripción

Como utilizar la funcion split para generar un array de strings a partir de un string.

Metodo

Si tenemos un string que contiene varios elementos con un separador comun (una coma por ejemplo) podemos generar un array de la siguiente manera:

string nombres = "Maria,Jose,Juan,Sandra";

string[] nombreArray = nombres.Split(',');

foreach (string nombre in nombreArray)
{
    Console.WriteLine(nombre);
}

Solo tenemos que llamar a la funcion Split indicando el separador, en este caso la coma, la función se encarga de generar un array con cada uno de los elementos resultantes al aplicar el separador.
