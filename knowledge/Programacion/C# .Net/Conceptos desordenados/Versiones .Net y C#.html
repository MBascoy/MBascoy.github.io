Consultar SDKs y Runtimes instalados

Para ver cuales son las versiones del SDK que tenemos instaladas ejecutamos el siguiente comando:

> dotnet --list-sdks

Para ver cuales son las versiones del Runtime que tenemos instaladas ejecutamos el siguiente comando:

> dotnet --list-runtimes
