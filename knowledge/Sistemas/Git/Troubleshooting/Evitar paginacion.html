Evitar paginacion

Descripcion

Cuando ejecutamos un comando como git diff o git branch a veces se puede producir paginacion incluso aunque el resultado sea inferior al de una pagina entera de la consola.

Solucion

Para evitar la paginacion cuando la salida sea inferior a una pagina entera de la consola podemos configurar git con el siguiente comando:

git config --global --replace-all core.pager "less -F -X"
