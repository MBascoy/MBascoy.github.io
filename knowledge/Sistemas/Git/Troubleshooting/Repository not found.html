Repository not found

Descripcion

Al intentar hacer un PULL o un PUSH obtenemos el siguiente error:

ERROR: Repository not found.
Solucion

Este error se puede dar porque el usuario que se esta utilizando para hacer el pull o el push no tiene permisos para leer/escribir en el repositorio.
