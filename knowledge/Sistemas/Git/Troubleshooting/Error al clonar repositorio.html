Error al clonar repositorios

Descripcion

Al intentar clonar un repositorio se produce el siguiente error:

error: invalid path 'Miscelanea/Expresion TRUE || TRUE && FALSE.html'
fatal: unable to checkout working tree
Metodo

En mi caso este error se produjo por los caracteres "|" en el nombre del archivo, ya que ese archivo inicialmente estaba en linux y al intentar hacer el clonado en windows surgia ese error ya que los nombres de archivos en windows no permiten esos caracteres.

Solucion
Editar el nombre del fichero para que contenga caracteres validos tanto en linux como en windows
