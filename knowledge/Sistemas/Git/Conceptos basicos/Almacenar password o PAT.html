Almacenar password o PAT

Descripcion

Como almacenar el usuario y contraseña o el personal access token (PAT), para no tener que meterlo seguido.

Metodo

Desde el terminal introducimos el siguiente comando:

> git config --global credential.helper store

La siguiente vez que se nos pida el usuario y la contraseña (o el PAT) se quedaran almacenados en el archivo ~/.git-credentials

Podemos listar los valores de configuracion con el siguiente comando:

> git config --global --list

NOTA: Tambien se puede usar --system en vez de global (En el caso de que no veamos el parametro definido entro de global)

Asi podemos verificar si esta bien configuradas las opciones, si el credential.helper esta configurado como cache o manager no se usuara el almacen de contraseñas
