Abortar merge

Descripcion

Como abortar un merge cuando tenemos conflictos pendientes de ser resueltos.

Metodo

Cuando realizamos un merge y obtenemos conflictos podemos volver al estado anterior y descartar la resolución de conflictos con el siguiente comando:

> git merge --abort
Abort stash pop

Cuando hacemos un stash pop a veces también se pueden dar conflictos, pero en este caso no podemos utilizar el comando "git merge --abort" en caso de que queramos abortar, tendremos que ejecutar el siguiente comando:

git reset --merge
