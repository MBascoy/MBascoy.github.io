Cambiar idioma de Git

Mbascoy.dev / Git / Cambiar idioma de Git.html
Descripcion

Por defecto Git coge el idioma del sistema operativo, podemos cambiarlo de la siguiente manera.

Metodo Linux

Para cambiar el idioma modificamos la variable de entorno LC_ALL, pero solo para git

LC_ALL=en_GB git

Para evitar tener que poner la variable de entorno siempre delante del comando podemos incluir la siguiente linea en nuestro .bash_rc:

alias git='LC_ALL=en_GB git'
