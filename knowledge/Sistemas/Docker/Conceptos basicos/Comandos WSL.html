Comandos

Listar imagenes de WSL:

wsl -l -v

Eliminar imagen (Distribucion) de WSL:

wsl --unregister Ubuntu

Establecer imagen por defecto:

wsl --set-default Ubuntu

Establecer version por defecto de WSL para las nuevas distribuciones instaladas:

wsl --set-default-version 2
