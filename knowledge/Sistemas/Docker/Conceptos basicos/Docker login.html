Comandos

Para hacer login con una cuenta en docker hub lo hacemos con el siguiente comando:

docker login

Una vez introducido el comando se nos preguntará usuario y contraseña

Para desloguearnos lo hacemos con el siguiente comando:

docker logout

Comprobar si estamos logueados

Para saber si estamos logueados con alguna cuenta ejecutamos el siguiente comando:

docker info | grep Username

Si tenemos alguna sesión iniciada su mostrará una linea Username con el usuario con el que estamos logueados.

NOTA: podemos ejecutar el comando docker info sin el grep para obtener mas información general sobre docker, como el registro contra el que estamos logueados.
