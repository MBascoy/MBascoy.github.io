Volumenes en DOcker

Comandos

Crear un volumen:

docker volume create <NOMBRE_VOLUMEN>

Listar volumenes:

docker volume ls

Borrar volumen

docker volume rm <NOMBRE_VOLUMEN>

La información de los volumenes de docker se almacenan en /var/lib/docker/volumes/NOMBRE_VOLUMEN/_data, si queremos borrar la información de un volumen podemos hacerlo borrando dicha carpeta:

# rm -rf /var/lib/docker/volumes/NOMBRE_VOLUMEN/_data
