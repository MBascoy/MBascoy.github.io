Montar sistema de archivos de windows

Descripcion

Como montar una unidad (como C:) dentro de wsl

Solución

Para realizar un montaje de la unidad C: lo haríamos de la siguiente manera:

sudo mount -t drvfs C: /mnt/c

Otra manera es editar el archivo /etc/fstab con la siguiente linea:

C: /mnt/c drvfs defaults 0 0

Si usamos esta segunda forma tendremos que ejecutar mount -a para que se recargue la configuración del fichero y se monte la unidad.
