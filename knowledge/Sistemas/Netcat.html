Netcat

Binario Windows

Netcat 1.12 Windows

Comandos

Conectarse a un puerto

nc -vv <IP> <PUERTO>

Ejemplo:

nc -vv 127.0.0.1 8080

Poner a la escucha

nc -vv -l -p <PUERTO>

Ejemplo:

nc -vv -l -p 8080
Peticiones HTTP

Solicitud:

GET / HTTP/1.1

Respuesta:

Content-Length: 52
Content-Type: text/html

<html>
<body>
<h1>Hello, World!</h1>
