Azure DevOps hacer PullRequest

Método

Para crear una PullRequest en Azure Devops nos vamos a la sección Pull requets y hacemos click en New pull request si hemos hecho un push de una rama recientemente nos saldrá una recomendación para hacer una pull request de la rama de la que acabamos de hacer push.

Seleccionamos desde que rama a que rama queremos hacer la pull request:

Indicamos un titulo para la pull request y hacemos click en crear:

Una vez creada la pull request, podemos aceptarla entrando en la propia pull request:

En principio nosotros mismos no deberíamos de tener permisos para aceptar nuestras propias pull requests, en caso de no tener permiso no aparecerá la opción Complete, en su lugar veremos esto:

Si no tenemos permisos solo veremos la opción Mark as draft, esta opción solo nos permite devolver la pull request a modo borrador para poder modificarla, pero nunca podremos aceptarla, de eso se encargaría el arquitecto del proyecto.
