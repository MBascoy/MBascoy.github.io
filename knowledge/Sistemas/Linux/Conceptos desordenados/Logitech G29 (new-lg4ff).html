Logitech G29 (new-lg4ff)

Descripcion

Instalación del driver new-lg4ff y como configurar el rango maximo de giro

Repositorio new-lg4ff
Driver new-lg4ff

Para instalar el driver seguimos lo siguientes pasos:

  • Descargamos el repositorio en /usr/src/:
    • $ cd /usr/src # git clone https://github.com/berarma/new-lg4ff.git
  • Instalamos dkms:
    • # zypper in dkms
  • Instalamos el driver:
    • # dkms install /usr/src/new-lg4ff
  • Actualizamos initramfs:

    • Este comando normalmente es:

    update-initramfs -u

    Pero en openSuSE es:

    # dracut -v -f
  • Reiniciamos el sistema y el driver ya estará instalado
Configurar rango maximo de giro

Para configurar el rango maximo de giro lo hacemos con el siguiente comando:

# echo 187 > /sys/bus/hid/drivers/logitech/{ID}/range

El parámetro del echo son los grados maximos de giro (187 en este caso), y el {ID} varía cada vez que se conecta el volante, simplemente podemos listar los archivos de la carpeta logitech y veremos cual es el id, tiene un formato como el siguiente: 0003:046D:C24F.0005

NOTA: este comando hay que hacerlo como root, a veces no es suficiente usar sudo.
