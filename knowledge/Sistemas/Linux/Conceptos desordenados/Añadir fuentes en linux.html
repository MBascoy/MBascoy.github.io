Añadir fuentes en Linux

Descripcion

Pasos para añadir nuevas fuentes en Linux

Resumen

Copiamos el archivo .ttf de la fuente dentro de /usr/share/fonts o de ~/.local/share/fonts (para el usuario local)

Si tenemos algun programa abierto lo reiniciamos para que recarge las fuentes del sistema

Enlaces
  • https://freefontsdownload.net/
  • https://fonts.google.com/
Metodo
  • Primero descargamos el archivo de la fuente
  • Creamos una carpeta (Solo para tenerlo todo mas ordenado) dentro de /usr/share/fonts o de ~/.local/share/fonts (para el usuario local):
    • mkdir "/usr/share/fonts/dream unicorn"
  • Copiamos el archivo comprimido de la fuente a esa carpeta (la carpeta que creamos dentro de /usr/share/fontsy lo descomprimimos:
    • cp dream-unicorn.zip /usr/share/fonts/dream\ unicorn/ cd /usr/share/fonts/dream\ unicorn/
    unzip dream-unicorn.zip

Listo, la fuente ya esta lista para usar, si teniamos algun programa abierto probablemente tendremos que reiniciar el programa para que recarge la lista de fuentes disponibles en el sistema

En realidad lo unico que hay que hacer es copiar el archivo .ttf a la carpeta /usr/share/fonts, no hace falta crear ninguna carpeta, pero asi se mantiene todo mas ordenado, en el caso de que la fuente tenga varios subtipos de fuentes
