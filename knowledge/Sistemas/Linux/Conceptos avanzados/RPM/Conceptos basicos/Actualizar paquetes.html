Actualizar paquetes

Descripcion

Como realizar la actualizacion de paquetes rpm que ya tenemos instalados en nuestro sistema pero tenemos un nuevo rpm de una version posterior.

Cuando tenemos un archivo .rpm que es una version actualizada de un paquete que ya tenemos instalada en nuestro sistema al intentar instalarlo veremos un error como el siguiente:

En este caso estamos intentando instalar el paquete de google-chrome con la version 96, pero ya tenemos instalada la version 93 y como vemos en la imagen los archivos de los paquetes entran en conflicto

Metodo

Para actualizar un paquete tendremos que usar la opcion -U del comando rpm:

> rpm -U <nombre archivo> > rpm -U google-chrome-stable_current_x86_64.rpm
