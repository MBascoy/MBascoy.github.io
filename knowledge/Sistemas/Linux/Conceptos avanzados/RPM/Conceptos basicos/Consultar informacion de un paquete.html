Consultar informacion de un paquete

Descripcion

Como consultar informacion de los paquetes .rpm, para poder saber su version u otros datos relacionados con el paquete.

Metodo

Para consultar informacion de un paquete rpm podemos hacerlo con el siguiente comando cuando es un archivo .rpm situado en un lugar concreto de nuestro sistema de archivos:

> rpm -qip <nombre archivo>

Podemos usar el siguiente comando cuando es un paquete rpm que ya tenemos instalado en nuestro sistema:

> rpm -qi <nombre paquete>

Para poder consultar el listado de todos los paquetes instalados en nuestros sistema podemos usar el siguiente comando:

> rpm -qa

NOTA: Podemos usar grep para buscar un paquete en la lista
