Verificar paquetes

Descripcion

Como verificar que todos los archivos de un paquete están instalados de manera correcta.

Metodo

Para verificar los arhivos de un paquete usamos el siguiente comando:

> rpm -V <nombre paquete>

Si todos los paquetes están exactamente igual que cuando se instaló el paquete el comando no dara ninguna salida, en el caso de que haya archivos modificados o archivos que han sido eliminados veremos una salida como la siguiente:

Si queremos restaurar todos los archivos al estado inicial de cuando instalamos el paquete ejecutamos el siguiente comando:

rpm -i --replacepkgs autofirma-1.7.1-1.noarch_SUSE.rpm

Básicamente es reinstalar el paquete.

Para hacer la verificación de archivos no necesitamos el archivo .rpm, pero para hacer la reinstalación si que es necesario el paquete .rpm
