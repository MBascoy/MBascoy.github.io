Vim no borra caracteres

Mbascoy.dev / Linux / Vim / Vim no borra caracteres
Descripcion

Cuando pulsamos la tecla Backspace vim no borra caracteres, o no borra los saltos de linea

Solución

Para resolver este problema tenemos que incluir la siguiente configuración en nuestro archivo vimrc:

set backspace=indent,eol,start
