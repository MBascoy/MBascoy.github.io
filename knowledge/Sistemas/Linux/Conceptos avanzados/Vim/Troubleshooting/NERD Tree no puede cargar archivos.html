NERD Tree no puede cargar archivos

Descripcion

Al lanzar NERD Tree se nos muestra el siguiente error:

NERDTree: 1 file(s) could not be loaded into the NERD tree

El error no impide el uso del plugin, pero nos informa de que hay un archivo no se muestra en la lista

Solución

Para saber cual es el archivo que da problemas tendremos que comparar la lista de archivos que aparecen en el listado de NERD Tree (Tenemos que activar los archivos ocultos con Shift + i) y la salida que nos puede dar un comando como "ls -la"

En este caso el archivo es request.txt y el motivo es que no tiene permisos para que NERD Tree pueda leer dicho archivo (podemos ver a la izquierda que no tiene absolutamente ningún permiso):
