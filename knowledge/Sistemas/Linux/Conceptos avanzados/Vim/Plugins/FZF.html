FZF

Descripcion

Nos permite realizar busquedas de archivos desde vim

Instalacion
  • Configuración vimrc:
    • Plug 'junegunn/fzf', { 'do': { -> fzf#install() } }
Plug 'junegunn/fzf.vim'
  • Dependencias

    • Para utilizar este plugin necesitamos los siguientes paquetes instaldos en nuestro sistema:

    • fzf: este comando debería de instalarse por defecto al instalar el plugin
    • ripgrep: este paquete es necesario para el comando RG que nos permite hacer busquedas en el contenido de los archivos

      • Podemos instalarlo con:

      zypper in ripgrep apt-get install ripgrep
    • silver searcher: este paquete nos permite utilizar el comando Ag

      • Podemos instalarlo con:

      zypper in the_silver_searcher apt-get install silversearcher-ag
    • bat: este paquete nos permite tener syntax highlight en las previews

      • Podemos instalarlo con:

      zypper in bat apt-get install bat
  • Mapeo de teclas:
    • nnoremap <C-f> :GFiles<CR> nnoremap <C-g> :Rg<CR>
Configuración

Comandos de busqueda

Si queremos buscar todos los archivos de la carpeta en la que estemos usamos el siguiente comando:

:Files

Si queremos tambien podemos hacer la busqueda de manera que se ignore todo lo que tenemos en el archivo .gitignore:

:GFiles

Configuracion de layout

Si queremos podemos modificar el aspecto de la interfaz de busqueda añadiendo una de las siguientes configuraciónes en nuestro .vimrc:

Hacer que la ventana de preview ocupe un 30% del ancho:

let g:fzf_preview_window = ['right,30%', 'ctrl-/']

Poner la ventana de preview en la parte superior y oculta hasta que se active con Ctrl + /:

let g:fzf_preview_window = ['up:40%:hidden', 'ctrl-/']
