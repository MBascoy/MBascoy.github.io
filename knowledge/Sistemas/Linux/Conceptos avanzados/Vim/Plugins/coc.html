coc

Descripcion

Nos permite utilizar un servidor LSP para tener autocompletación de código y otras funciones como "go to definition".

Instalacion
  • Configuración vimrc:
    • Plug 'neoclide/coc.nvim', {'branch': 'release'}
  • Para instalar los diferentes paquetes para cada lenguaje lo hacemos con el siguiente comando:
    • :CocInstall <paquete-lenguaje>
    • HTML:
      • :CocInstall coc-html
    • JavaScript/TypeScript
      • :CocInstall coc-tsserver
    • Java
      • :CocInstall coc-java
    • Json
      • :CocInstall coc-json
