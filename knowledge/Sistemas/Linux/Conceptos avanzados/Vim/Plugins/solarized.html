solarized

Descripcion

Nos permite cambiar el esquema de colores de vim.

Instalacion
  • Configuración vimrc:
    • Plug 'altercation/vim-colors-solarized'
  • Copiamos el esquema de colores del plugin que se acaba de descargar en la carpeta de colores de vim:

    • (Si no tenemos la carpeta .vim/colors tenemos que crearla)

    mkdir ~/.vim/colors mv ~/.vim/plugged/vim-colors-solarized/colors/solarized.vim ~/.vim/colors/
  • Editamos el archivo .vimrc para que se utilice el esquema de colores de solarized usando la siguiente configuracion:
    • syntax enable
set background=dark
colorscheme solarized
  • Con estos cambios el plugin ya estaría funcional en terminales como gvim, pero si estamos usando un terminal de 256 colores, tendremos que añadir la siguiente configuración:

    • NOTA: esta configuración tiene que ir ANTES de colorscheme solarized

    let g:solarized_termcolors=256

    Esta configuración usará una versión degradada de los colores de solarized, si queremos usar los colores puros del plugin, tendremos que utilizar una terminal modificada o una interfaz gráfica como gvim, o modificar los colores de la terminal para que cuadre con el esquema de colores de solarized

Configuracion terminales

Para usar el esquema de colores desde una terminal sin tener que usar el sistema de colores degragado tendremos que añadir a mayores configuraciones en la terminal que vayamos a usar.
