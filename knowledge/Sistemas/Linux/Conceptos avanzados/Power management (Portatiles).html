Como configurar el governor scaler para que un portatil est√© en modo ahorro o modo de rendimiento

Metodo
  • Consultar los modos disponibles:
    • cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_available_governors
  • Consultar modo activo actualmente:
    • cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor
  • Establecer un nuevo modo, performance en este caso (no permanente):
    • echo performance | sudo tee /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor

cpufrequtils

Con el paquete cpufrequtils tambien podemos hacer estas comprobaciones e incluso hacer los ajustes permanentes

NOTA: En algunos sistemas hay un paquete similar que se llama cpupower que funciona exactamente igual.

  • Instalar (en Debian):
    • apt-get install cpufrequtils
  • Consultar modos disponibles:
    • cpufreq-info -g
  • Consultar modos activo actualmente:
    • cpufreq-info -p
  • Establecer nuevo modo:
    • cpufreq-set -g performance

    Si queremos que los cambios sean permanentes tenemos que modificar el archivo /etc/default/cpufrequtils indicando el siguiente contenido:

    GOVERNOR='performance'
