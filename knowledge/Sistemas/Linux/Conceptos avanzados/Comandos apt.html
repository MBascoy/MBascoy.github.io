Comandos apt

Descripcion

Comandos para el gestor de paquetes apt.

Comandos

Buscar paquetes:

apt-cache search <NOMBRE_PAQUETE>

Actualizar repositorios:

apt-get update

Mostrar paquetes con actualizaciones:

apt list --upgradeable

Actualizar todos los paquetes con actualizaciones disponibles:

apt-get upgrade

Actualizar un solo paquete con actualizaciones disponibles

apt-get upgrade <NOMBRE_PAQUETE>
