Instalar themes

Descripción

Cuando tenemos Gnome shell en verdad tenemos dos tipos de themes los de Gnome shell y los de GTK. En la página de Gnome look los themes de GTK3/4 suelen incluir la parte necesaria de Gnome shell

Los themes de Gnome shell cambian la apariencia de los elementos del escritorio tanto la barra lateral (donde están las aplicaciones de acceso rapido), la barra superior y la vista del menu de aplicaciones.

Los themes de GTK cambian la apariencia de las aplicaciones.

Descargar nuevos themes: Gnome look (desde la seccion GTK3/4)

Instalar themes

Una vez tenemos descargado un theme lo tenemos que extraer en una de estas dos carpetas (según queramos instalarlo solo para el usuario actual o para todos los usuarios):

  • ~/.themes
  • /usr/share/themes

Una vez lo hemos extraido tenemos que activarlo desde Gnome tweaks:

Tenemos que activar tanto la opción de aplicaciones como la de Gnome shell, aunque no todos los themes tienen ambas opciones.

NOTA: Si ya teniamos abierto gnome tweaks antes de extraer el theme tendremos que cerrar y volver a abrir la aplicación para que se recargen los themes
