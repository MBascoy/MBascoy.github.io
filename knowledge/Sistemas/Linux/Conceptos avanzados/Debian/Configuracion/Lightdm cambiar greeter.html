Lightdm cambiar greeter

Descripcion

Cambiar greeter de lightdm para cambiar la pantalla de login.

Metodo

Para cambiar el greeter tenemos que cambiar la siguiente linea de configuración en el archivo /etc/lightdm/lightdm.conf:

greeter-session=lightdm-gtk-greeter

Algunos de los greeters que podemos indicar en la configuración son los siguienes

  • lightdm-gtk-greeter (viene por defecto)
  • ukui-greeter (se instala con: apt-get install ukui-greeter)
  • slick-greeter (se instala con: apt-get install slick-greeter)
  • arctica-greeter (se instala con: apt-get install arctica-greeter)

NOTA: para usar el lector de huellas el que mejor funciona es el gtk-greeter, despues el arctica funciona bastante bien, y los otros funcionan fatal.
