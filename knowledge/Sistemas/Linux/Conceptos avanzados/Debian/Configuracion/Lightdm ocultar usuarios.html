Lightdm ocultar usuarios

Mbascoy.dev / Linux / Debian / Lightdm ocultar usuarios.html
Descripcion

Ocultar usuarios del listado cuando vamos a hacer login.

En este caso queremos ocultar el usuario citrixlog

Metodo

Para ocultar usuarios en el listado tenemos que hacer lo siguiente:

Nos vamos a la carpeta /var/lib/AccountsService/users/ aqui dentro creamos un archivo con el nombre del usuario que queremos ocultar, citrixlog en nuestro caso, y a√±adimos dentro el siguiente contenido:

[User]
SystemAccount=true

Podemos hacerlo directamente con este comando:

sudo bash -c "echo -e [User]\\\\nSystemAccount=true > /var/lib/AccountsService/users/citrixlog"

Una vez hecho esto el usuario ya no saldra en la lista al hacer login.
