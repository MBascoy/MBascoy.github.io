Descripcion

Como cambiar el display manager que tenemos configurado.

Configurar display manger XFCE

Metodo

Una manera rapida para cambiar el display manager es hacer un reconfigure de uno de los paquetes de un display manager:

# dpkg-reconfigure lightdm

o tambien:

# dpkg-reconfigure gdm3

Al hacer esto se nos lanzará una interfaz en la que se nos preguntará que display manager queremos tener activado.

Lightdm configurar desktop environment

Para seleccionar el Desktop environment que se va a lanzar (gnome, xfce, kde...etc) podemos hacerlo desde la propia interfaz grafica de lightdm arriba a la derecha:

Si queremos hacer el cambio desde el archivo de configuración se hace en el siguiente archivo:

~/.dmrc

El contenido del archivo es el siguiente para xfce:

[Desktop]
Session=xfce

Para Gnome:

[Desktop]
Session=gnome

Para Gnome classic:

[Desktop]
Session=gnome-classic
