Mbascoy.dev / Linux / Debian / Actualizar firmware (t480s)
Descripcion

A veces aparecen actualizaciones de firmware en el gestor de actualizaciones que no podemos instalar:

Al intentar instalarlas siempre obtenemos un error

Metodo

Para poder instalar dichas actualizaciones lanzamos el siguiente comando:

sudo fwupdmgr update
