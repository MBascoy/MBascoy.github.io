Metodo

Si estamos teniendo problemas con Gnome podemos reinstalarlo usando el siguiente comando:

sudo apt-get install --reinstall gnome-shell

Una vez tengamos reinstalado el entorno de gnome puede ser necesario tener que reinstalar ciertas aplicaciones como nautilus o alguna otra, el comando es el mismo pero con la aplicaci√≥n que queramos reinstalar

sudo apt-get install --reinstall nautilus
