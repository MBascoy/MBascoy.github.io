Descripcion

Para configurar la lista de repositorios lo hacemos desde el archivo:

/etc/apt/sources.list

El contenido del archivo con los repositorios mas elementales sería el siguiente (para debian bullseye)

deb http://ftp.es.debian.org/debian/ bullseye main
deb-src http://ftp.es.debian.org/debian/ bullseye main

deb http://security.debian.org/debian-security bullseye-security main contrib
deb-src http://security.debian.org/debian-security bullseye-security main contrib

deb http://ftp.es.debian.org/debian/ bullseye-updates main contrib
deb-src http://ftp.es.debian.org/debian/ bullseye-updates main contrib

deb http://deb.debian.org/debian bullseye main contrib non-free
deb-src http://deb.debian.org/debian bullseye main contrib non-free

deb http://deb.debian.org/debian-security/ bullseye-security main contrib non-free
deb-src http://deb.debian.org/debian-security/ bullseye-security main contrib non-free

deb http://deb.debian.org/debian bullseye-updates main contrib non-free
deb-src http://deb.debian.org/debian bullseye-updates main contrib non-free

deb http://ftp.de.debian.org/debian sid main
