flameshot

Mbascoy.dev / Linux / flameshot.html
Descripcion

Nos permite hacer capturas de pantalla y editarlas directamente

Metodo

Para instalar el paquete lo hacemos con:

apt-get install flameshot

Para lanzar la aplicacion lo hacemos con el siguiente comando:

flameshot gui

Si ya tenemos configurado una aplicación asociada a la tecla Prt Scr tenemos que quitar dicha configuración y añadir un nuevo atajo para que lance el comando anterior

En mi caso en Debian con Gnome, lo hago desde la configuración general:

Desactivar captura de pantalla del sistema

Crear nuevo shortcut para flameshot:

Indicamos el comando para el atajo:

Capturar pantalla completa

Una vez pulsamos la tecla Prt Scr ya podemos seleccionar una zona y utilizar las herramientas, pero si queremos simplemente hacer una captura de toda la pantalla lo hacemos pulsando directamente Ctrl + s para guardar en un archivo o Ctrl + c para guardar en el portapapeles.

Troubleshooting

Si a la hora de tomar capturas tenemos un retardo muy grande tenemos que editar el siguiente archivo:

/usr/share/applications/org.flameshot.Flameshot.desktop

En la linea de Exec eliminamos el argumento --delay 500 para eliminar el retardo de medio segundo

NOTA: Aún así al usar una tecla como shortcut tendremos algún tipo de retardo (si lo comparamos con la ejecución directa del comando), pero aplicando este metodo el retardo es mucho menor.
