NetworkManager vs Wicked

Mbascoy.dev / Linux / NetworkManager vs Wicked.html
Descripcion

Cambiar entre NetworkManager y Wicked como servicios de gestion de red.

Metodo

Para cambiar entre un gestor y otro lo hacemos con systemctl:

systemctl disable NetworkManager.service systemctl enable wicked.service

O viceversa

Configuracion

En el caso de wicked las configuraciones se almacenan en:

/etc/sysconfig/network

En el caso de NetworkManager en:

/etc/NetworkManager/system-connections
Troubleshooting

En el caso de NetworkManager si no nos detecta ninguna interfaz podemos intentar añadirla manualmente, en caso de que aún así no funcione podemos añadir el siguiente fichero:

En la ruta /etc/NetworkManager/conf.d/10-managed-devices.conf:

[main]
plugins=ifupdown,keyfile

[keyfile]
unmanaged-devices=*,except:type:wifi,except:type:wwan,except:type:ethernet

[ifupdown]
managed=true

Una vez añadido este archivo podemos intentar añadir de nuevo manualmente una interfaz ethernet.
