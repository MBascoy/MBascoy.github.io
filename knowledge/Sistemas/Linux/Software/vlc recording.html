vlc recording

Descripcion

Grabar streamings (canales IPTV) con vlc desde la linea de comandos (sin necesidad de tener vlc abierto)

Metodo

El siguiente comando nos permite grabar una retransmision:

cvlc -vvv ENLACE_RETRANSMISION --sout file/ts:MI_GRABACION.ts

La retransmision se grabaría en la carpeta actual con el nombre MI_GRABACION.ts, la grabación es infinita hasta que nosotros paramos la ejecución del comando con Cntrl + c

Grabación con temporizador

Si queremos podemos usar el siguiente comando para grabar un rango de tiempo:

sleep 5; cvlc -vvv ENLACE_RETRANSMISION --sout file/ts:MI_GRABACION.ts & sleep 100; killall vlc

Este comando retrasa el inicio del comando de grabación el tiempo que indiquemos en el primer sleep (5 segundos en este caso), y termina la grabación despues del tiempo que hayamos indicado en el segundo sleep (100 segundos en este caso), el tiempo que indicamos en el segundo sleep cuenta desde que se ejecuta el comando de grabación (o sea que sería el tiempo efectivo de grabacion).
