video codecs

Descripcion

Como instalar codecs de video en OpenSuse Tumbleweed para poder reproducir cualquier tipo de archivo

Metodo

Primero añadimos el repositorio de packman para tumbleweed

zypper addrepo -f http://ftp.gwdg.de/pub/linux/misc/packman/suse/openSUSE_Tumbleweed/ packman

NOTA: El repositorio de packman puede cambiar según la distribución que tengamos

Instalamos los siguiente paquetes:

zypper install ffmpeg-4 lame gstreamer-plugins-bad gstreamer-plugins-ugly gstreamer-plugins-ugly-orig-addon gstreamer-plugins-libav libavdevice57 vlc-codecs

Una vez instalados estos paquetes ya deberíamos de poder reproducir cualquier tipo de archivo de video (principlamente con VLC).

Listado paquetes

El listado de paquetes que instalamos es el siguiente (por si queremos buscarlos para otra distribución):

  • ffmpeg-4
  • lame
  • gstreamer-plugins-bad
  • gstreamer-plugins-ugly
  • gstreamer-plugins-ugly-orig-addon
  • gstreamer-plugins-libav
  • libavdevice57
  • vlc-codecs
