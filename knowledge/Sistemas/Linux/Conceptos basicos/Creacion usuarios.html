Descripcion

Crear y eliminar usuarios desde la linea de comandos

Metodo

Para crear un usuario lo hacemos con el siguiente comando:

# useradd -s /bin/bash -m NOMBRE_USUARIO

Para establecer una contrase√±a (necesario para hacer login) lo hacemos asi:

# passwd NOMBRE_USUARIO

Si queremos borrar el usuario y su carpeta personal lo hacemos con el siguiente comando:

# userdel -r NOMBRE_USUARIO
