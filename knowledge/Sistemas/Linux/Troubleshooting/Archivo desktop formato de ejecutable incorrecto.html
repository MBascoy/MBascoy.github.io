Archivo desktop formato de ejecutable incorrecto

Descripcion

Al intentar ejecutar un archivo .desktop (Archivo de escritorio de KDE) obtenemos el siguiente error

Explicacion

En este caso estamos intentando ejecutar el archivo autofirma.desktop:

Si nos fijamos en el contenido del .desktop vemos que intenta ejecutar el archivo /usr/bin/autofirma cuyo contenido es el siguiente

Como podemos ver el archivo NO es un ejecutable binario, si no que es un script (solo contiene una linea que ejecuta java, pero es un script al fin y al cabo), el error se produce porque no hay un interprete asignado a dicho script, para solucionarlo solo debemos aniadir #!/bin/bash al inicio del archivo, de manera que quedaria tal que asi:

#!/bin/bash
java -jar /usr/lib64/autofirma/autofirma.jar $*

Esto solucionaria el problema ya que la primera linea indica que el archivo debe de ser interpretado con /bin/bash

Error

Codigo de error 100 desconocido
execvp: Formato de ejecutable incorrecto
