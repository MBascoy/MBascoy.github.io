Agregar proyecto existente

Descripcion

Como añadir un proyecto ya existente a un repositorio TFVC.

Metodo
  • En el Team Explorer seleccionamos Open:
  • Seleccionamos el proyecto que queremos agregar al repositorio:
  • Una vez seleccionemos el proyecto este se abrirá en el Solution explorer, hacemos click derecho en la solución y despues en Add Solution to Source Control:
  • En la ventana que se nos abre podemos seleccionar a que repositorio queremos añadir el proyecto y a que workspace
  • Al finalizar, el nuevo proyecto ya saldrá en la lista de soluciones del repositorio:
  • Para terminar ya solo tenemos que hacer check in para sincronizar el nuevo proyecto con el repositorio remoto.
