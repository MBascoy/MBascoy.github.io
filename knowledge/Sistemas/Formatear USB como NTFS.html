Formatear USB como NTFS

Descripcion

Metodo para formatear un USB desde linux como NTFS para poder usarlo tanto en linux como windows con archivos mayores de 4 GB

Metodo

Seguiremos los siguientes pasos:

  • Usamos cfdisk para borrar todas las particiones que haya creadas en el dispositivo
    • # cfdisk /dev/sdb

    NOTA: Cuando ejecutamos cfdisk se le pasa como parametro la ruta del dispositivo (/dev/sdb) y NO la ruta de una particion (/dev/sdb1)

  • Creamos una nueva particion que ocupe todo el dispositivo
  • Asociamos el tipo 0x07 a la particion, es el tipo de particion para NTFS
  • Escribimos los cambios que hemos realizado en la tabla de particiones
  • Ya hemos terminado con la edicion de la tabla de particiones, podemos salir del cfdisk, ahora usamos mkfs.ntfs para formatear la particion (podemos usar la opcion -f para hacer formato rapido, pero si podemos mejor hacer el formato completo)

    • # mkfs.ntfs /dev/sdb1

    NOTA: Cuando ejecutamos mkfs.ntfs se le pasa como parametro la ruta de una particion (/dev/sdb1) y NO la ruta del dispositivo(/dev/sdb)

En este momento el dispositivo ya esta formateado y listo para ser montado y utilizado

Dar un nombre al dispositivo

Si queremos darle un nombre concreto al dispositivo de almacenamiento podemos hacerlo usando ntfslabel: # ntfslabel /dev/sdb1 Datos

Ahora al conectar nuestro dispositivo aparecera con el nombre que le hemos puesto
