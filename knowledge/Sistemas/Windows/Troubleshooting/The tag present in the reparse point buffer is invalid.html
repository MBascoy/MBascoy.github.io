The tag present in the reparse point buffer is invalid

Descripcion

Al intentar acceder a una carpeta obtenemos el siguiente error:

is not accesible The tag present in the reparse point buffer is invalid

Si intentamos borrarla simplemente no nos deja, no se produce ningún error, pero no se borra.

Solución

Para solucionar este error tenemos que abrir una consola (cmd) con permisos de administrador y ejecutar el siguiente comando:

chkdsk c: /f

NOTA: cambiar c: por el nombre de la unidad en el que se encuentre la carpeta corrupta.

Nos responderá con un mensaje diciendo que el disco está en uso y que si queremos programar una comprobación del disco para el siguiente reinicio respondemos que si (Y), y reiniciamos el sistema.

Al reiniciar el sistema se comprobará el disco y se solucionarán todos los problemas que haya en el sistema de archivos, una vez se inicie el sistema de nuevo ya prodriamos tener acceso a la carpeta que nos daba problemas.
