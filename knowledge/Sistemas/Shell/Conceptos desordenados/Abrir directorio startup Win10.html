Abrir directorio startup Win10

Mbascoy.dev / Shell / Abrir directorio startup Win10
Descripcion

Abrir la carpeta startup. donde se almacenan las aplicaciónes que se abren de manera automática al iniciar sesión. del usuario actual.

Metodo

Desde Windows > Run (tecla Win + R) ejecutamos el siguiente comando:

shell:startup
