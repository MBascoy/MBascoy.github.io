Consultar y conectar directorios de red

Descripcion

Como manejar unidades de red en Windows 10.

Metodo

Consultar

Para consultar que unidades de red tenemos actualmente y en que estado se encuentran usamos el siguiente comando:

> net use

Eliminar

Para borrar una unidade de red usamos el siguiente comando:

> net use h: /DELETE

Con este comando borrariamos la unidad h:

Conectar nueva unidad

> net use h: \\example.com\recurso$\carpeta

Con este comando conectariamos la carpeta \carpeta del recurso recurso$ del servidor example.com y lo asignar√≠amos a la unidad h:
