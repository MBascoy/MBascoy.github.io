Finder crear actions

Descripcion

Como crear Quick Actions para utilizar con los archivos en Finder.

Metodo

Usaremos Automator para crear las Quick Actions.

Seguimos los siguientes pasos:

  • Abrimos Automator y seleccionamos Quick Action:
  • En la parte superior "Workflow receives current" files or folders "in" Finder:
  • En el menu de la izquierda buscamos "open finder" y arrastramos el action Open Finder Items a la zona de actions:
  • Ahora en la seccion que aparece en "Open with" seleccionamos la aplicación con la que queremos abrir el archivo/carpeta seleccionado:

    • NOTA: La aplicación tiene que estar en la carpeta "Applications" si no no aparecerá ni siquiera en la opción de "Other"

  • Elegimos la App, en este caso usaré el VSCode:
  • En la sección superior en Image podemos elegir un icono personalizado para la accion, en este caso elegiré el de VSCode:
  • Y ahora solo tenemos que irnos a File > Save

    • El nombre que le demos al guardar será el nombre que aparecerá en la lista de acciones.

Una vez hecho todo esto la accion ya aparecerá en Finder en el menú contextual de Quick Actions:

NOTA: Las Quick Actions se almacenan en ~/Library/Services/ si queremos borrarlas tendremos que hacerlo manualmente desde esa carpeta.

Actions con Shell Script

Si queremos podemos crear Actions que lancen un comando de consola.

En este caso crearé una Action que copie la ruta del archivo seleccionado.

El proceso es igual que el anterior, pero en vez de añadir la Action "Open Finder Items" añadiremos Run Shell Script:

Ahora en la configuración de la Action cambiamos arriba a la derecha para indicar "Pass input" as arguments, y ponemos el siguiente código:

echo "$1" | tr '\n' '\0' | pbcopy

Guardamos y ya tendremos la Quick Action disponible.
