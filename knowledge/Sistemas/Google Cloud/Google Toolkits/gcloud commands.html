gcloud commands

Comandos

Obtener cuentas disponibles y cuenta actualmente activa de Google Cloud:

gcloud auth list

Obtener lista de proyectos que tiene nuestra cuenta actual:

gcloud projects list

Obtener proyecto configurado por defecto:

gcloud config get project

Lista de cuentas en un proyecto y que roles tienen:

gcloud projects get-iam-policy <ID PROYECTO>
