Consultar password Wi-Fi

Mbascoy.dev / Consultar password Wi-Fi.html
Descripcion

Como consultar la contraseña del wifi a la que estámos conectados desde la consola de windows

Explicacion

Primero ejecutamos el siguientes comando para saber todas las conexiones que tenemos configuradas en nuestro sistema:

netsh wlan show profile

A continuación ejecutamos el siguiente comando indicando la red de la que queremos conocer la contraseña:

netsh wlan show profile name="SWO_Santiago_LinkSyS" key=clear

La constraseña figura en la seccion Key Content.
