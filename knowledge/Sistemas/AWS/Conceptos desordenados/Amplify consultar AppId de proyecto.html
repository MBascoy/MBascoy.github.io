Amplify consultar AppId de proyecto

Mbascoy.dev / AWS / Amplify consultar AppId de proyecto.html
Descripcion

Consultar estado y AppId que está asociada a un proyecto.

Metodo

Para ver el estado de un proyecto lo hacemos con el siguiente comando:

amplify status -v

Si tenemos cambios pendientes de ser sincronizados este comando nos lo mostrará.

Consultar AppId

Si queremos consultar el AppId de una App de amplify lo podemos ver en el siguiente archivo:

amplify/backend/amplify-meta.json

O tambien:

amplify/#current-cloud-backend/amplify-meta.json
