Descripcion

Como cargar un script sql en una base de datos MySQL usando DBeaver

Explicacion

Primero nos conectamos a la base de datos:

Seleccionamos el tipo de base de datos, mysql5 en este caso:

Indicamos los datos de conexion a la base de datos, direccion (localhost si es local), usuario y contraseña:

En el panel de la izquierda podemos ver como se ha creado la conexion con la base de datos:

Ahora pasamos a cargar el script en la base de datos, primero vamos a SQL Editor > New SQL Script:

Seleccionamos nuestra base de datos:

Antes de importar el script nos aseguramos de que nuestra base de datos es la que está seleccionada:

Nos vamos a SQL Editor > Import SQL Script:

Seleccionamos nuestro script:

Seleccionamos la opcion "Execute SQL Script" (La opcion que se muestra en la imagen)

Una vez hecho esto nuestro script ya está cargado, si no se nos muestra la base de datos en nuestra conexion simplemente hacemos click derecho encima de Databases y click en Refresh
