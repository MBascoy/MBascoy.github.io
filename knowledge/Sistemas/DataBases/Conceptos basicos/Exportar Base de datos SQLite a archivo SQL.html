Exportar Base de datos SQLite a archivo SQL

Descripcion

Como generar un archivo .sql a partir de una base de datos SQLite

Metodo

Abrimos nuestra base de datos con SQLite:

sqlite3 crud_example.db

Indicamos el archivo donde se guardará el .sql:

.output archivo_exportado.sql

Exportamos la base de datos:

.dump

Salimos del interprete de SQLite

.exit

Si queremos podemos exportar solo una tabla, para ello al hacer el dump lo haríamos indicando el nombre de la tabla:

.dump pedidos
