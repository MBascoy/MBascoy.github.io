SQLite crear base de datos a partir de script

Mbascoy.dev / DataBases / SQLite crear base de datos a partir de script
Descripcion

Como crear una base de datos SQLite a partir de una base de datos ya exportada en un archivo .sql

Pagina web SQLite

Explicacion

Para este ejemplo usaremos la base de datos NorthwindSQLite.sql

Ejecutamos el siguiente comando:

sqlite3 dbtest.db -init NorthwindSQLite.sql

El primer parámetro es el nombre del archivo en el que se guardará la base de datos que se creará, el segundo parámetro es el nombre del script que se usará para generar la base de datos.

La creación de la base de datos puede llevar un rato, una vez finalice el comando entra directamente en la linea de comandos de SQLite podemos salir pulsando cntrl + D, no es necesario ejecutar ninguna comando mas en la linea de comandos.

NOTA: SQLite no necesita de un servicio para manejar la base de datos, simplemente se maneja a traves del archivo de base de datos generado, en este caso dbtest.db.
