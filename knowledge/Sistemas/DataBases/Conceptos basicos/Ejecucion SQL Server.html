Ejecucion SQL Server

Descripcion

Como administrar la ejecución del SQL Server en Windows

Explicacion

Para administrar el proceso del SQL Server en windows buscamos en el menú de unicio: SQL Server 2019 Configuration Manager

Otra manera de acceder a la misma configuración es usando el siguiente comando desde la consola:

C:\Windows\SysWOW64\mmc.exe /32 C:\WINDOWS\SysWOW64\SQLServerManager15.msc

NOTA: Se necesitan permisos de administrador para gestionar estos procesos, si no tenemos permisos de administrador podemos entrar igual a la ventana de configuración pero cuando intentemos parar un proceso no surtirá ningún efecto (no se muestra ningún mensaje de error, simplemente no se hace la operación).

Al acceder a la configuración veremos la siguiente ventana:

Desde aquí podemos iniciar o parar el servidor SQL Server
