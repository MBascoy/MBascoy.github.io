Instalacion SQL Server

Descripcion

Como instalar SQL Server version Developer en Windows

Explicacion
  • Descargas el instalador de SQL Server developer (https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
  • Ejecutas el instalador y esperas a que termine la descarga (necesitas permisos de administrador para ejecutar el instalador)
  • Cuando se descargue todo se abre el SQL Server Installation Center, haces click en New SQL Server stand-alone installation or add features to an existing installation
  • Seleccionas la opción Developer
  • (te aparece una lista de posibles problemas que puedes tener, si te pone algo del firewall en amarillo no pasa nada)
  • Aceptas los terminos de licencia
  • En Feature Selection eliges Database Engine Services y le das a siguiente
  • En Instance Configuration elige Default instance y le das a next.
  • En Server Configuration, asegurarse de que SQL Server Database Engine, esté configurado para iniciarse automáticamente. Establecer SQL Server Browser para que se inicie automáticamente tambien.
  • Despues en Database Engine Configuration en la tabla de Server Configuration eliges como Authentication Mode pones Mixed , pones una contraseña en las cajas de texto y despues eliges la opcion Add para añadir un usuario, tienes que buscar tu usuario y darle a check names (no le deas a add current user porque te añadirá el usuario de la persona de soporte)
  • En la última pantalla pulsa Install
  • Una vez finaliza la instalación le das a Close
  • Despues en el SQL Server Installation Center en Installation haces click en Install SQL Server Management Tools
  • Se te abre el navegador, te bajas el SSMS (la version 18 creo) (SQL Server Management Studio)
  • Ejecutas el instalador (Necesotas permisos de administrador nuevamente)
  • Cuando acabe la instalación reinicias el PC y ya tendrías todo listo
