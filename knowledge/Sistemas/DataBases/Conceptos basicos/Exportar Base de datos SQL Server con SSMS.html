Exportar Base de datos SQL Server con SSMS

Descripcion

Como exportar una base de datos ya sea solo el esquema o con todos los datos de las tablas.

Explicacion

Para exportar una base de datos en SSMS, hacemos click derecho encima de la base de datos que queramos exportar y seleccionamos Tasks > Generate Scripts :

Se nos abrirá un wizard para la exportación, pulsamos next:

En la siguiente ventana dejamos marcada la opción de exportar la base de datos completa, en caso de que queramos exportar solo algunas tablas concretas marcamos la otra opción y en la lista seleccionamos el contenido que queremos exportar:

En la siguiente ventana seleccionamos Save as script file e indicamos donde se guardará el archivo final.

A mayores en esta página podemos hacer click en el boton Advanced para indicar que datos queremos exportar (Si solo la estructura de la base de datos o tambien todos los datos de las tablas)

Al pulsar Advanced veremos la siguiente ventana, en la opción Types of data to script podemos indicar si queremos solo es esquema, solo los datos o el esquema y los datos:

En la siguiente página se nos muestra un resumen de las opciones de exportación:

Al llegar a la última página se empieza a realizar la exportación, esperamos hasta que termine y una vez finalizado el proceso pulsamos Finish:

El archivo .sql ya estará generado en la localización que le hayamos indicado.
