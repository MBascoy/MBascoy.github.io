PowerShell variables de entorno

Descripcion
  • Ver todas las variables de entorno:
    • dir env:
  • Declarar una variable de entorno:

    • Sintaxis: $env:<VARIABLE> = <VALOR>

    Ejemplo:

    $env:DOCKER_HOST = 'tcp://localhost:2376'
  • Consultar el valor de una sola variable:

    • Sintaxis: $env:<VARIABLE>

    Ejemplo:

    $env:DOCKER_HOST
