Como abrir un ticket a soporte tecnico para realizar instalacion de software

Resumen

Accedemos al siguiente enlace:

Solicitar Instalacion de software

Rellenamos el formulario, lo enviamos y esperamos a que nos contacte el tecnico de soporte

Estado de solicitud

En el siguiente enlace podemos ver las solicitudes que tenemos abiertas (tambien las cerradas) con soporte tecnico, si accedemos a las solicitudes podemos enviar mensajes a la gente de soporte tecnico en relacion con el ticket que tenemos abierto para avisar de a que hora estaremos disponibles por ejemplo.

Soporte tecnico tickets abiertos
Metodo

Primero accedemos al One-Hub:

SWO One-Hub

A continuacion nos vamos a All Tools:

Despues nos vamos a ServiceNow:

Ahora pinchamos en IT Support

Despues accedemos a Request Something New

En la siguiente pantalla en el menu de la izquierda nos vamos a Modern Workplace

Y dentro de esta categoria nos vamos a Applications Management

Y en las opciones que nos aparecen seleccionamos Install New Software/Application

A continuacion en el formulario rellenamos toda la informacion indicando lo que queremos instalar y pulsamos Submit

Ya solo queda esperar a que nos contacte el tecnico para conectarse a nuestro ordenador de manera remota e instalar lo que le pedimos

Durante el proceso que el tenico este instalando el software tenemos que estar presentes en el ordenador y tener todos los archivos que necesitemos instalar a mano, el tecnico practicamente lo que hara sera poner la clave de administrador para poder instalar el software, todo lo demas depende de nosotros
