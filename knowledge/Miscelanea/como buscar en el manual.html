Descripcion

Como realizar una busqueda de contenido dentro del manual usando el bash de linux

Metodo

Para realizar una busqueda podemos hacer una busqueda del archivos (buscando una palabra contenida en el nombre del archivo) o podemos hacer una busqueda dentro del contenido de los propios archivos del manual (lo que suele ser mas optimo para encontrar algo)

Lo primero siempre es estar dentro de la carpeta del manual:

  • Abrimos un shell
  • Nos ponemos dentro de la carpeta del manual > cd knowledge

busqueda por nombre de archivo:

> find ./ -iname "*busqueda*"

busqueda por contenido de archivo:

> grep -ri "busqueda"

Sustituyendo la palabra "busqueda" por el criterio que queramos buscar
