Configurar AP para extender Wi-Fi

Descripcion

Como configurar varios puntos de acceso para crear una única red wifi con mucho alcance.

Se trata de crear una sola red (con un solo nombre), no varias redes con varios nombres.

Metodo

Para crear una única red con varios AP tenemos que configurar los AP tal que así:

  • Mismo ESSID
  • Misma contraseña
  • Mismo tipo de seguridad (WPA/WPA2)
  • Canales que no se solapen
