Configurar AP para extender Wi-Fi

Descripcion

Como configurar varios puntos de acceso para crear una única red wifi con mucho alcance.

Se trata de crear una sola red (con un solo nombre), no varias redes con varios nombres.

Metodo

Para crear una única red con varios AP tenemos que configurar los AP tal que así:

  • Mismo ESSID
  • Misma contraseña
  • Mismo tipo de seguridad (WPA/WPA2)
  • Canales que no se solapen

    • En la siguiente imagen podemos ver cuales son los canales no solapantes tanto para la banda de 2.4 como para la de 5 Ghz:

    A continuación se muestra la configuración de mis dos AP:

    Access Point 1:

    • 2.4Ghz
      • SSID: p3xzk
      • Seguridad: WPA2/WPA Personal
      • Canal: 11
      • Modo: 802.11b/g/n
      • Channel width: 40 Mhz
    • 5Ghz
      • SSID: p3xzk5Ghz
      • Seguridad: WPA2/WPA Personal
      • Canal: 140
      • Modo: 802.11a/n/ac
      • Channel width: 80 Mhz

    Access Point 2:

    • 2.4Ghz
      • SSID: p3xzk
      • Seguridad: WPA2/WPA Personal
      • Canal: 6
      • Modo: 802.11b/g/n
      • Channel width: 20/40 Mhz
    • 5Ghz
      • SSID: p3xzk5Ghz
      • Seguridad: WPA2/WPA Personal
      • Canal: 100
      • Modo: 802.11a/n/ac
      • Channel width: Auto 20/40/80 Mhz

Una vez tengamos los dos AP configurados de esta manera todos los dispositivos deberían de saltar de uno a otro de manera seamless la conexión no se debería de perder, simplemente la cobertura de wifi debería de aumentar automáticamente una vez te alejas de un AP y te acercas al otro.
