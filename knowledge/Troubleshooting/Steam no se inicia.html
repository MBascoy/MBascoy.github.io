Steam no se inicia

Mbascoy.dev / Steam no se inicia
Descripcion

Steam no se inicia una vez ejecutado el cliente, normalmente se queda atascado en la pantalla de "conectando a la cuenta"

Metodo

Lo primero que podemos hacer es probar a borrar una serie de carpetas dentro del directorio de steam:

  • Entramos a la carpeta de steam
    • cd ~/.local/share/Steam
  • Borramos la carpeta Appcache
    • rm -r appcache
  • Intentamos arrancar steam de nuevo, si continuan los problemas borramos las carpetas config y userdata
    • rm -r config rm -r userdata
  • Intentamos arrancar steam de nuevo, si todavia no arranca movemos la carpeta de steam al completo tal que asi:
    • cd ~/.local/share/ mv Steam Steamold
  • Una vez hecho esto ejecutamos Steam y deberia de realizar una instalacion nueva por completo (El cliente incluso descargara de nuevo todos los archivos necesarios).

    NOTA: si Steam todavia no arranca, probablemente sea un problema no relaccionado directamente con el cliente de STEAM, podriamos intentar ejecutar steam desde la consola para ver si lanza algun error concreto.

    • Si hemos llegado al ultimo paso para arreglar el arranque de Steam tendremos que mover de nuevo la carpeta steamapps de nuestra carpeta steamold para no tener que descargar de nuevo todos los juegos que ya teniamos descargados

    cd ~/.local/share/ rm -r Steam/steamapps/ mv Steamold/steamapps/ Steam/
