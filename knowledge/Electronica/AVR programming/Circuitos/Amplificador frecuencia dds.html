Circuito para amplificar una señal dds.

Si queremos amplificar una señal dds (Direct Digital Synthesis), podemos hacerlo de igual manera que con una señal cuadrada usando un transistor, pero en este caso tendremos que usar un condensador cerámico (100 nf) para generar un low pass filter y hacer así que el altavoz obtenga la onda generada y no las señales individuales que usamos para generar la onda.

Si no ponemos el condensador cerámico el altavoz emitirá un sonido estridente coincidiendo con las señales que hemos generado usando el PWM, esto no ocurre al conectar el altavoz directamente al microcontrolador porque la intensidad de salida es muy baja, pero al conectar el transistor el altavoz recibe una intensidad mucho mas alta permitiendole reaccionar a cada señal individual que hemos generado, lo que generará ese sonido estridente en vez de la onda que queremos generar usando dds.
